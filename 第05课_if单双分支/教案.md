### 教学目标

1. **理解"条件判断"的概念**——能用自己的话说出"如果……就……"在程序里怎么写。
2. **掌握 if 单分支和 if-else 双分支的语法**——能独立写出语法正确、花括号配对的分支程序。
3. **熟练使用六种比较运算符**（`>` `<` `>=` `<=` `==` `!=`），能把生活中的判断条件翻译成代码。

## 二、教学重点与难点

### 重点

- 比较运算符的含义与写法（尤其 `==` 和 `=` 的区别）。
- if 单分支的语法格式。
- if-else 双分支的语法格式。
- 花括号 `{}` 的配对规则。

### 难点

- `=`（赋值）与 `==`（判断相等）的区分——这是初学者最高频错误。
- 理解"条件为真时执行、为假时跳过"的程序执行流程。
- 多条语句时忘记加花括号导致逻辑错误。

## 四、讲稿提纲

> 以下用 **四年级学生能听懂的语言** 组织，教师可根据实际节奏自由展开。

### 4.1 复习导入（10 min）

**开场提问：**

> 同学们，前几节课我们学会了让电脑做加减乘除运算。可是你们想想——每天早上起床，你做的第一个"运算"是什么？
>
> 其实不是算数，而是 **做选择**！
>
> - **如果**今天下雨，**就**带伞；
> - **如果**今天很冷，**就**穿厚外套；**否则**穿薄外套。
>
> 电脑也需要做选择！今天我们就来教它"如果……就……"。

**关键比喻——岔路口：**

> 想象你走在一条路上，前面出现了一个 **岔路口**。路口有一块牌子写着一个条件，比如"身高超过 120cm"。
>
> - 如果你满足条件，就走左边的路（执行某段代码）；
> - 如果不满足，就走右边的路（执行另一段代码），或者直接走大路继续往前。
>
> 这就是今天要学的 **if 分支**！

### 4.2 比较运算符（10 min）

**引入：**

> 岔路口的牌子上写的条件，需要做"比较"才能判断。比如"分数 >= 60"就是在比较。C++ 有六种比较运算符：

**板书/PPT：**

| 运算符 | 含义 | 举例 | 结果 |
|--------|------|------|------|
| `>` | 大于 | `5 > 3` | 真（true） |
| `<` | 小于 | `5 < 3` | 假（false） |
| `>=` | 大于等于 | `5 >= 5` | 真 |
| `<=` | 小于等于 | `3 <= 5` | 真 |
| `==` | 等于（判断） | `5 == 3` | 假 |
| `!=` | 不等于 | `5 != 3` | 真 |

**重点强调（至少说三遍）：**

> ⚠️ **一个等号 `=` 是"赋值"（把右边的值放进左边的变量）；两个等号 `==` 才是"判断相等"！**
>
> 就像生活中——
> - `=` 是"把苹果放进篮子里"（赋值动作）。
> - `==` 是"篮子里是不是苹果？"（提问判断）。

### 4.3 if 单分支（15 min）

**语法模板：**

```
if (条件) {
    条件成立时执行的代码;
}
```

**生活翻译：**

> "如果考试成绩 >= 90 分，就输出'优秀！'" ——翻译成 C++：

```cpp
if (score >= 90) {
    cout << "优秀！" << endl;
}
```

**流程图（教师在黑板上画）：**

```
        ┌──────────┐
        │ 条件成立？ │
        └────┬─────┘
           / \
       是 /   \ 否
         /     \
  ┌──────┐     │
  │执行代码│     │
  └──┬───┘     │
     │         │
     └────┬────┘
          ↓
      继续往下走
```

**比喻加深理解：**

> 单分支就像一条路上有个"彩蛋房间"——满足条件的人可以进去拿奖品，不满足的人直接走过去，不影响后面的路。

**花括号规则：**

> 花括号 `{` 和 `}` 就像一对括号朋友，**必须成对出现**。它们圈住的代码就是"满足条件才做的事"。
>
> 🔑 小技巧：**写完 `if ()` 立刻先把 `{}` 写好，再回来往里面填代码**，这样就不会忘记配对！

### 4.4 if-else 双分支（15 min）

**语法模板：**

```
if (条件) {
    条件成立时执行的代码;
} else {
    条件不成立时执行的代码;
}
```

**生活翻译：**

> "如果考试成绩 >= 60 分，就输出'及格'；否则输出'不及格'。"

```cpp
if (score >= 60) {
    cout << "及格" << endl;
} else {
    cout << "不及格" << endl;
}
```

**比喻：**

> 双分支就是真正的 **岔路口**——左边或右边，你必须选一条走，不可能两条都走，也不可能两条都不走。

**强调 else 的位置：**

> `else` 要紧跟在 if 的 `}` 后面，不能单独出现。`else` 不需要写条件，它的意思就是"其他所有不满足 if 条件的情况"。

### 4.5 综合讲解要点

- 花括号内只有一条语句时，花括号可以省略——**但我们约定：初学阶段永远写花括号**，养成好习惯，避免出错。
- if-else 执行完后，程序继续往下走后面的代码，不会停在那里。
- 条件部分的小括号 `()` 不能省略。

## 五、课堂例题

### 例题 1：判断正数（if 单分支入门）

> **题目：** 输入一个整数，如果它是正数，输出"positive"。

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    if (n > 0) {
        cout << "positive" << endl;
    }
    return 0;
}
```

**讲解要点：**
- `n > 0` 就是条件，写在小括号里。
- 只有当 `n > 0` 为真时，才会输出 "positive"。
- 如果输入 0 或负数，程序什么都不输出，直接结束。

---

### 例题 2：奇偶判断（if-else 双分支入门）

> **题目：** 输入一个整数，判断它是奇数还是偶数。

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    if (n % 2 == 0) {
        cout << "even" << endl;
    } else {
        cout << "odd" << endl;
    }
    return 0;
}
```

**讲解要点：**
- 复习取余运算符 `%`：`n % 2` 的结果要么是 0（偶数），要么是 1（奇数）。
- `n % 2 == 0` 注意是**两个等号**，不是一个。
- if-else 保证"偶数"和"奇数"一定会输出其中一个，不会两个都输出或都不输出。

---

### 例题 3：考试等级（双分支 + 生活场景）

> **题目：** 输入一个考试成绩（0-100 的整数）。如果 >= 60 分，输出 "PASS"；否则输出 "FAIL"。

```cpp
#include <iostream>
using namespace std;

int main() {
    int score;
    cin >> score;
    if (score >= 60) {
        cout << "PASS" << endl;
    } else {
        cout << "FAIL" << endl;
    }
    return 0;
}
```

**讲解要点：**
- 条件中用 `>=` 而不是 `>`，因为 60 分也算及格。
- 让学生自己思考：如果把 `>=` 改成 `>`，输入 60 会怎样？

---

### 例题 4：比较两个数的大小（综合应用）

> **题目：** 输入两个整数 a 和 b，输出其中较大的那个数。

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    if (a > b) {
        cout << a << endl;
    } else {
        cout << b << endl;
    }
    return 0;
}
```

**讲解要点：**
- 这里 a == b 时会走 else 分支，输出 b——因为两个相等，输出哪个都对。
- 引导思考：如果要求"当相等时输出两个都行"，代码该怎么改？（为下节课 if-else if 做铺垫。）

---

### 例题 5：密码验证（== 和 = 的区分实战）

> **题目：** 设定密码为 123456。让用户输入一个数，如果等于密码就输出 "Welcome!"，否则输出 "Wrong password!"。

```cpp
#include <iostream>
using namespace std;

int main() {
    int password = 123456;
    int input;
    cin >> input;
    if (input == password) {
        cout << "Welcome!" << endl;
    } else {
        cout << "Wrong password!" << endl;
    }
    return 0;
}
```

**讲解要点：**
- **故意演示错误版本**：把 `==` 写成 `=`，运行程序看看会发生什么——不管输入什么都显示 "Welcome!"。
- 借此加深印象：**`=` 是赋值，`==` 是判断**。

### 练习 4：谁是赢家？（比较 + 双分支）

> 两个小朋友比赛跑步，输入他们各自的用时（整数，单位秒）。用时少的人赢。输出赢家的用时。如果用时一样，输出 "tie"。

**提示：** 这道题需要考虑三种情况（a < b、a > b、a == b），可以先用两个 if-else 嵌套试试看。也可以暂时用两个 if 来写，下节课学了 if-else if 后再优化。

**参考答案：**

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    if (a < b) {
        cout << a << endl;
    } else {
        if (a > b) {
            cout << b << endl;
        } else {
            cout << "tie" << endl;
        }
    }
    return 0;
}
```

> 教师说明：这里出现了 if 的"嵌套"——在 else 里面又写了一个 if-else。下节课会专门学习这种写法，今天先感受一下就行。

## 七、常见错误与应对

### 错误 1：把 `==` 写成 `=`

```cpp
// ❌ 错误写法
if (n = 5) {       // 这是赋值！n 被改成 5，条件永远为"真"
    cout << "hello";
}

// ✅ 正确写法
if (n == 5) {      // 这才是判断 n 是否等于 5
    cout << "hello";
}
```

**应对方法：** 反复强调口诀——**"一个等号是赋值，两个等号是判断"**。可以让学生在纸上写 5 遍。

---

### 错误 2：忘记写花括号导致逻辑错误

```cpp
// ❌ 错误写法：只有第一句受 if 控制
if (score >= 60)
    cout << "及格" << endl;
    cout << "恭喜你！" << endl;  // 这句不管条件是否成立都会执行！

// ✅ 正确写法
if (score >= 60) {
    cout << "及格" << endl;
    cout << "恭喜你！" << endl;  // 现在两句都在花括号里了
}
```

**应对方法：** 强制要求学生**任何时候都写花括号**，形成肌肉记忆。教口诀——**"写完 if，先打花括号"**。

---

### 错误 3：条件的小括号写错或漏掉

```cpp
// ❌ 错误
if n > 0 {           // 缺少小括号
if (n > 0)（         // 用了中文括号

// ✅ 正确
if (n > 0) {         // 英文小括号 + 英文花括号
```

**应对方法：** 提醒学生写代码时**切换到英文输入法**，养成习惯。

---

### 错误 4：else 后面加了条件

```cpp
// ❌ 错误
if (n > 0) {
    cout << "正数";
} else (n <= 0) {     // else 后面不能加条件！
    cout << "非正数";
}

// ✅ 正确
if (n > 0) {
    cout << "正数";
} else {               // else 就是"否则"，不需要条件
    cout << "非正数";
}
```

**应对方法：** 告诉学生——**else 的意思是"除了上面 if 的情况之外的所有情况"，不需要再写条件。** 想写条件的话，那是下节课要学的 `else if`。

---

### 错误 5：花括号不配对

```cpp
// ❌ 错误：少了一个 }
if (n > 0) {
    cout << "正数";

// 编译器会报错！
```

**应对方法：** 教学生养成 **"先写一对花括号，再往里填代码"** 的习惯。在 Dev-C++ 等 IDE 中可以开启括号匹配高亮。
