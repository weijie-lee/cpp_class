### 教学目标（可检验）

1. **能说出**冒泡排序的基本原理：相邻两个元素比较，大的往后交换，每一轮"冒"出一个最大值。
2. **能手动模拟**一组 5 个数的完整冒泡排序过程，正确写出每一轮结束后数组的状态。
3. **能独立编写**冒泡排序的完整 C++ 程序，对一组不超过 100 个整数从小到大排序并输出。

---

## 二、教学重点与难点

### 重点
- 交换两个变量的值（temp 临时变量法）
- 冒泡排序的核心逻辑：双重循环 + 相邻比较交换
- 手动模拟排序过程，建立直觉

### 难点
- 外层循环和内层循环的边界（为什么外层 `n-1` 轮、内层 `n-1-i` 次）
- 理解"每一轮确定一个最大值的位置"这一不变性
- 下标越界问题（`a[j]` 和 `a[j+1]` 的边界）

## 四、讲稿提纲

### 4.1 导入：为什么需要排序？（0′ – 8′）

> 🎙️ 老师话术参考：

"同学们，假设你们班刚考完试，老师拿到了全班 40 个人的成绩，想找出谁是第一名——如果成绩单是**乱的**，你得一个一个看，好累啊！但如果成绩从高到低**排好了**，一眼就能看出来谁是第一名、谁是最后一名。"

"再举个例子：你去图书馆找一本书，如果书架上的书是**随便放的**，你可能找一下午都找不到。但如果按拼音排好了，很快就能找到。"

"所以**排序**就是把一堆乱七八糟的东西，按照某种规则（比如从小到大、从矮到高）排成整齐的一列。今天我们要学的，就是程序员排序最经典的入门方法——**冒泡排序**！"

**互动提问**：
- "生活中还有什么地方需要排序？"（等学生回答：考试排名、身高排队、跑步比赛名次……）

### 4.2 新知一：交换两个变量的值（8′ – 18′）

> 🎙️ 老师话术参考：

"排序的核心动作就是**交换**。比如排队的时候，小明 140cm 站前面，小红 135cm 站后面，老师说矮的站前面——他俩就得**换个位置**。"

"那在程序里，怎么交换两个变量呢？比如 `a = 3`，`b = 5`，我们想让 `a` 变成 5，`b` 变成 3。"

"有同学可能会说：`a = b; b = a;` ——这样行不行？（停顿等学生思考）"

"不行！因为第一步 `a = b` 之后，`a` 已经变成 5 了，原来的 3 就**丢了**！`b = a` 得到的还是 5。"

#### 三杯水比喻 🥤

"想象桌上有两个杯子：A 杯装了**橙汁**，B 杯装了**牛奶**。你想让 A 杯装牛奶、B 杯装橙汁，怎么办？"

"你需要一个**空杯子 C**！"
1. 先把 A 杯的橙汁倒进 C 杯（C = 橙汁）
2. 再把 B 杯的牛奶倒进 A 杯（A = 牛奶）
3. 最后把 C 杯的橙汁倒进 B 杯（B = 橙汁）

"搞定！这个空杯子在程序里就叫 **temp（临时变量）**。"

```
int temp = a;   // 空杯子先存 a 的值
a = b;          // 把 b 的值给 a
b = temp;       // 把空杯子里的值给 b
```

**板书/PPT 动画**：画三个杯子，演示交换过程。

### 4.4 新知二：冒泡排序原理（25′ – 45′）

#### 第一层比喻：水里的气泡 🫧

> "你们见过水里的气泡吗？小气泡从水底一直往上**浮**，越大的气泡浮得越高。冒泡排序就像这样——**大的数字像气泡一样，一步一步往后面冒**。"

#### 第二层比喻：排队比身高 👫

> "想象一排同学站好了，老师从左边开始，**每次只看相邻的两个人**：如果左边的比右边的**高**，就让他俩**换个位置**。然后再看下一对……一直看到最右边。走完一趟之后，**最高的那个人一定到了最右边**！"

> "然后老师再从左边开始走第二趟，这次不用管最右边那个了（他已经是最高的），又能把第二高的放到倒数第二的位置……一直走下去，就排好啦！"

#### 🎮 互动活动：真人冒泡排序（约 8 分钟）

**准备**：请 5 个同学站到讲台前，每人举一张写有数字的纸牌（例如：**5, 2, 8, 1, 4**）。

**过程**：
1. 老师当"指针"，站在最左边两个人中间。
2. 比较左右两人的数字，如果左边大就让他俩换位置，否则不动。
3. 老师往右移一步，再比较……直到走到头。
4. 问同学："现在最右边是谁？（8）最大的到位了！"
5. 重复第二轮、第三轮……直到排好。

**关键提问**：
- "第一轮走完，最大的数字到哪了？"
- "第二轮需不需要再看最后一个？为什么？"
- "一共需要走几轮？"

#### 手动推演（纸上模拟）

在黑板/PPT 上写出数组：`[5, 2, 8, 1, 4]`

```
原始: [5, 2, 8, 1, 4]

第1轮:
  比较 5 和 2 → 5>2, 交换 → [2, 5, 8, 1, 4]
  比较 5 和 8 → 5<8, 不换 → [2, 5, 8, 1, 4]
  比较 8 和 1 → 8>1, 交换 → [2, 5, 1, 8, 4]
  比较 8 和 4 → 8>4, 交换 → [2, 5, 1, 4, 8]  ← 8 到位！

第2轮:
  比较 2 和 5 → 不换 → [2, 5, 1, 4, 8]
  比较 5 和 1 → 交换 → [2, 1, 5, 4, 8]
  比较 5 和 4 → 交换 → [2, 1, 4, 5, 8]  ← 5 到位！

第3轮:
  比较 2 和 1 → 交换 → [1, 2, 4, 5, 8]
  比较 2 和 4 → 不换 → [1, 2, 4, 5, 8]  ← 4 到位！

第4轮:
  比较 1 和 2 → 不换 → [1, 2, 4, 5, 8]  ← 2 到位！

排序完成！1, 2, 4, 5, 8
```

#### 总结规律（引导学生说出）

- `n` 个数，需要 **`n - 1` 轮**。
- 第 `i` 轮（从 0 开始），需要比较 **`n - 1 - i` 次**（因为后面 `i` 个已经到位了）。
- 每次只看**相邻两个**，大的往后换。

## 五、课堂例题

### 例题 1：基础冒泡排序（由浅入深 ⭐）

**题目**：输入 n 个整数，用冒泡排序把它们从小到大排好，然后输出。

**输入样例**：
```
5
5 2 8 1 4
```

**输出样例**：
```
1 2 4 5 8
```

#### 讲解思路

> "我们就按刚才在黑板上模拟的过程写代码。外面一层循环控制**轮数**，里面一层循环控制每一轮**比较几次**。每次比较相邻的两个，如果左边比右边大，就用 temp 交换。"

#### 完整代码

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[105];  // 数组开大一点，养成好习惯
    // 读入数据
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    // 冒泡排序
    for (int i = 0; i < n - 1; i++) {        // 外层：n-1 轮
        for (int j = 0; j < n - 1 - i; j++) { // 内层：每轮比较的次数
            if (a[j] > a[j + 1]) {            // 如果左边比右边大
                int temp = a[j];              // 交换（三杯水）
                a[j] = a[j + 1];
                a[j + 1] = temp;
            }
        }
    }

    // 输出排好的数组
    for (int i = 0; i < n; i++) {
        if (i > 0) cout << " ";  // 数字之间用空格隔开
        cout << a[i];
    }
    cout << endl;

    return 0;
}
```

#### 逐行讲解要点

| 代码行 | 讲解 |
|--------|------|
| `int a[105];` | 数组大小开大一点（比 n 的最大值大几个），防止越界 |
| `i < n - 1` | n 个数只需要 n-1 轮，最后一个自动到位 |
| `j < n - 1 - i` | 每轮结束后最后面的 i 个数已经排好了，不用再比 |
| `a[j] > a[j+1]` | 只在左边**大于**右边时才交换（等于不交换） |
| `int temp = ...` | 交换三步走：存→覆盖→还原 |

### 例题 2：输出每一轮的排序过程（进阶 ⭐⭐）

**题目**：输入 n 个整数，用冒泡排序从小到大排序。要求**每完成一轮**，输出当前数组的状态。

**输入样例**：
```
5
5 2 8 1 4
```

**输出样例**：
```
第1轮: 2 5 1 4 8
第2轮: 2 1 4 5 8
第3轮: 1 2 4 5 8
第4轮: 1 2 4 5 8
```

#### 讲解思路

> "这道题跟例题 1 几乎一模一样，只是每一轮结束后，我们多加一个循环把当前的数组打印出来。这样同学们就能**看到排序的过程**，跟我们刚才在黑板上推演的一模一样！"

#### 完整代码

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[105];
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (a[j] > a[j + 1]) {
                int temp = a[j];
                a[j] = a[j + 1];
                a[j + 1] = temp;
            }
        }
        // 每一轮结束后打印当前数组
        cout << "第" << i + 1 << "轮: ";
        for (int k = 0; k < n; k++) {
            if (k > 0) cout << " ";
            cout << a[k];
        }
        cout << endl;
    }

    return 0;
}
```

#### 讲解要点

- 输出语句放在**外层循环里面、内层循环外面**——也就是"每完成一整轮之后"打印。
- `i + 1` 是因为 `i` 从 0 开始，显示给人看要从第 1 轮开始。
- 这道题的价值：帮助理解排序过程，也是调试的好方法——排序出了问题，打印中间过程找 bug。

### 例题 3：从大到小排序（变式 ⭐⭐）

**题目**：输入 n 个整数，用冒泡排序把它们**从大到小**排序后输出。

**输入样例**：
```
4
3 1 7 2
```

**输出样例**：
```
7 3 2 1
```

#### 讲解思路

> "跟从小到大只有**一个地方**不同——你们猜猜是哪里？"（等学生思考）
>
> "对了！就是比较的那个符号！从小到大是大的往后冒（`>`），从大到小就是**小的往后冒**（`<`）。只需要把 `a[j] > a[j+1]` 改成 `a[j] < a[j+1]` 就行了！"

#### 完整代码

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[105];
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    // 冒泡排序（从大到小）
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (a[j] < a[j + 1]) {   // 注意这里改成了 <
                int temp = a[j];
                a[j] = a[j + 1];
                a[j + 1] = temp;
            }
        }
    }

    for (int i = 0; i < n; i++) {
        if (i > 0) cout << " ";
        cout << a[i];
    }
    cout << endl;

    return 0;
}
```

## 七、常见错误与应对

### 错误 1：交换时没用临时变量

```cpp
// ❌ 错误写法
a[j] = a[j + 1];
a[j + 1] = a[j];   // 此时 a[j] 已经被覆盖了！
```

> **应对**：回到"三杯水"比喻——"你直接把橙汁倒掉，再想倒回来，杯子已经空了呀！一定要先用空杯子存起来。"

### 错误 2：循环边界写错

```cpp
// ❌ 常见错误：内层写成 j < n - 1
for (int j = 0; j < n - 1; j++)  // 每轮都比较 n-1 次，没有减 i

// ❌ 常见错误：内层写成 j <= n - 1 - i
for (int j = 0; j <= n - 1 - i; j++)  // 多比了一次，a[j+1] 越界！
```

> **应对**：
> - 第一种不会出错，只是多做了无用功（也可以先这样写，理解后再优化）。
> - 第二种会越界，引导学生用**手动推演**检查边界：`j` 最大是多少？`j+1` 会不会超过数组范围？

### 错误 3：忘记读入 n 或数组

```cpp
// ❌ 忘记读入
int a[105];
for (int i = 0; i < n - 1; i++) { ... }  // n 没有赋值，是随机数！
```

> **应对**：提醒学生养成习惯——"输入、处理、输出"三步走，先把输入写完再写排序。

### 错误 4：输出格式问题

```cpp
// ❌ 多了末尾空格
for (int i = 0; i < n; i++) {
    cout << a[i] << " ";  // 最后一个数后面也有空格
}
```

> **应对**：教两种写法——
> 1. `if (i > 0) cout << " ";` 在数字前面加空格（跳过第一个）
> 2. 特判最后一个不加空格

### 错误 5：数组开太小

```cpp
int a[5];  // 如果 n 可能到 100，这就不够了！
```

> **应对**："数组就像书包，宁可背个大的空着，也别背个小的塞不下。一般题目说 n ≤ 100，我们就开 105。"

## 九、下节课预告

> 🔮 **第 20 课：选择排序与插入排序**

"今天我们学了冒泡排序——相邻两个比，大的往后冒。下节课我们要学**另外两种**排序方法：

- **选择排序**：每次从剩下的数里**选出最小的**，放到前面。就像你从一堆扑克牌里每次挑最小的抽出来。
- **插入排序**：一张一张摸牌，每摸到一张就**插到手牌中正确的位置**。就像你打扑克时整理手牌一样！

学完三种排序之后，你就能理解排序的核心思想了。后面我们还会学一个超级好用的 `sort` 函数，一行代码搞定排序——但只有理解了原理，才能在竞赛中灵活运用哦！"

> - 例题 3 和练习 3、4 作为弹性内容，学有余力的同学完成即可
