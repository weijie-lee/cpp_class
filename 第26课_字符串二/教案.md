### 4.2 讲解一：拼接与比较（15 分钟）

#### A. 字符串拼接：`+`

> 想象你有两列小火车，一列写着 "Hello"，一列写着 "World"。把它们**连在一起**，就变成 "HelloWorld"。在 C++ 里，用加号 `+` 就能做到！

```cpp
string a = "Hello";
string b = "World";
string c = a + b;     // c 是 "HelloWorld"
string d = a + " " + b; // d 是 "Hello World"
```

> **注意：** 也可以拼一个字符上去哦！

```cpp
string s = "abc";
s = s + 'd';   // s 变成 "abcd"
s += 'e';      // s 变成 "abcde"，更简洁的写法
```

**随堂演示1：** 读入姓和名，输出完整姓名。

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string xing, ming;
    cin >> xing >> ming;
    cout << xing + ming << endl;
    return 0;
}
```

#### B. 字符串比较：字典序

> 你们查字典的时候怎么比较两个单词谁在前面？先看第一个字母，a 在 b 前面；如果第一个字母一样，就看第二个……这就叫**字典序**。
>
> C++ 里比较字符串用的 `<`、`>`、`==` 就是按字典序来的！

```cpp
string s1 = "apple", s2 = "banana";
cout << (s1 < s2) << endl;   // 输出 1（true），因为 a 在 b 前面
cout << (s1 == s2) << endl;  // 输出 0（false）
```

> ⚠️ **小陷阱：** `"apple" < "banana"` 是对的，但 `"9" > "10"` 也是对的！为什么？因为字典序逐个字符比，`'9'` > `'1'`，所以 "9" > "10"。这跟数字大小不一样哦！

**随堂演示2：** 读入两个单词，按字典序从小到大输出。

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string a, b;
    cin >> a >> b;
    if(a <= b)
        cout << a << " " << b << endl;
    else
        cout << b << " " << a << endl;
    return 0;
}
```

### 4.3 讲解二：substr 与 find（15 分钟）

#### A. 截取子串：`s.substr(pos, len)`

> 还记得我们的小火车吗？`substr` 就是从火车的**第 pos 节车厢开始**，往后**截 len 节**出来，组成一列新的小火车。
>
> 特别注意：pos 从 **0** 开始数！

```cpp
string s = "HelloWorld";
// 下标:     0123456789
cout << s.substr(0, 5) << endl;  // 输出 "Hello"（从第0个开始，截5个）
cout << s.substr(5, 5) << endl;  // 输出 "World"（从第5个开始，截5个）
cout << s.substr(3, 4) << endl;  // 输出 "loWo"（从第3个开始，截4个）
```

> 如果只写一个参数 `s.substr(5)`，意思就是从第 5 个一直截到最后！

```cpp
cout << s.substr(5) << endl;  // 输出 "World"
```

**随堂演示3：** 读入一个字符串和两个整数 pos、len，输出截取的子串。

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string s;
    int pos, len;
    cin >> s >> pos >> len;
    cout << s.substr(pos, len) << endl;
    return 0;
}
```

#### B. 查找子串：`s.find("xxx")`

> `find` 就像在火车上**找人**。你告诉它你要找的内容，它告诉你**第一次出现的位置**（下标）。如果找不到，它会返回一个特殊的值 `string::npos`。

```cpp
string s = "I love coding";
cout << s.find("love") << endl;    // 输出 2（从下标2开始出现）
cout << s.find("hate") << endl;    // 找不到，输出一个超大数字
```

> 判断找没找到的标准写法：

```cpp
if(s.find("love") != string::npos){
    cout << "找到了！" << endl;
} else {
    cout << "没找到……" << endl;
}
```

**随堂演示4：** 读入一个句子和一个单词，判断句子中是否包含该单词。

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string sentence, word;
    getline(cin, sentence);
    cin >> word;
    if(sentence.find(word) != string::npos)
        cout << "Yes" << endl;
    else
        cout << "No" << endl;
    return 0;
}
```

### 4.4 讲解三：遍历字符串与 ASCII（20 分钟）

#### A. 用循环遍历字符串

> 我们的小火车有 `s.size()` 节车厢，编号从 0 到 `s.size()-1`。要**检查每节车厢**，就用 for 循环一节一节地走过去！

```cpp
string s = "hello";
for(int i = 0; i < s.size(); i++){
    cout << s[i] << " ";
}
// 输出：h e l l o
```

#### B. ASCII 码——字符的"身份证号"

> 每个字符在计算机里都有一个**数字编号**，就像每个同学都有学号一样。这个编号系统叫 **ASCII 码**。
>
> 你们只需要记住三个"明星学号"：
> - **`'A'` = 65**，`'B'` = 66，……，`'Z'` = 90
> - **`'a'` = 97**，`'b'` = 98，……，`'z'` = 122
> - **`'0'` = 48**，`'1'` = 49，……，`'9'` = 57
>
> 发现了吗？大写字母和小写字母之间差了 **32**！`'a' - 'A' = 32`。
>
> 字符和整数可以互相转换，这是一个**超级有用**的技巧！

```cpp
char c = 'A';
cout << (int)c << endl;      // 输出 65
cout << (char)(c + 32) << endl; // 输出 'a'（大写变小写）
```

> 判断一个字符是不是大写字母：

```cpp
if(c >= 'A' && c <= 'Z'){
    cout << "是大写字母！" << endl;
}
```

#### C. 实用应用1：统计某字符出现次数

> 目标：数一数字符串里有多少个 `'a'`。

**随堂演示5：**

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string s;
    cin >> s;
    int cnt = 0;
    for(int i = 0; i < s.size(); i++){
        if(s[i] == 'a') cnt++;
    }
    cout << cnt << endl;
    return 0;
}
```

#### D. 实用应用2：大小写转换

> 把一个全是小写字母的字符串变成全大写——每个字符减 32 就行！

**随堂演示6：**

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string s;
    cin >> s;
    for(int i = 0; i < s.size(); i++){
        if(s[i] >= 'a' && s[i] <= 'z'){
            s[i] = s[i] - 32;  // 小写变大写
        }
    }
    cout << s << endl;
    return 0;
}
```

> 💡 小技巧：也可以用 `toupper(s[i])` 和 `tolower(s[i])`，效果一样！

#### E. 实用应用3：字符串翻转

> 把 "hello" 变成 "olleh"——就像把小火车**掉个头**！
>
> 方法一：从后往前遍历，把字符拼到新字符串上。
> 方法二：首尾交换，就像排队时第一个和最后一个换位置，第二个和倒数第二个换位置……

**随堂演示7（方法一）：**

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string s;
    cin >> s;
    string t = "";          // 准备一列空火车
    for(int i = s.size() - 1; i >= 0; i--){
        t += s[i];          // 从后往前，一节一节接上去
    }
    cout << t << endl;
    return 0;
}
```

**随堂演示7（方法二，首尾交换）：**

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string s;
    cin >> s;
    int left = 0, right = s.size() - 1;
    while(left < right){
        swap(s[left], s[right]);
        left++;
        right--;
    }
    cout << s << endl;
    return 0;
}
```

> 🎁 彩蛋：其实 C++ 自带了一个 `reverse` 函数哦！`reverse(s.begin(), s.end());` 一行搞定！

## 五、课堂例题（由浅入深）

### 例题1：最长的名字（难度 ★☆☆）

**题意：** 读入 n 个人的名字（英文，无空格），输出最长的那个名字。如果长度相同，输出先读入的那个。

**思路讲解：**
> 这道题就像比赛"谁的火车最长"。我们用一个变量记住"目前最长的火车"，每读入一个新的，就比一比，如果比它还长就换掉。

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int n;
    cin >> n;
    string best = "";        // 目前最长的名字
    for(int i = 0; i < n; i++){
        string name;
        cin >> name;
        if(name.size() > best.size()){
            best = name;     // 发现更长的，更新！
        }
    }
    cout << best << endl;
    return 0;
}
```

**输入示例：**
```
3
Alice
Bob
Catherine
```
**输出示例：**
```
Catherine
```

### 例题2：字符串中数字之和（难度 ★★☆）

**题意：** 读入一个字符串（可能包含字母和数字），求所有**数字字符**对应的数值之和。例如 `"a1b2c3"` 的数字之和为 `1+2+3=6`。

**思路讲解：**
> 遍历每节车厢，检查这个字符是不是数字（`'0'` 到 `'9'`）。如果是，怎么把字符变成数字呢？用 `s[i] - '0'`！因为 `'3' - '0' = 51 - 48 = 3`。

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string s;
    cin >> s;
    int sum = 0;
    for(int i = 0; i < s.size(); i++){
        if(s[i] >= '0' && s[i] <= '9'){
            sum += s[i] - '0';   // 字符转数字
        }
    }
    cout << sum << endl;
    return 0;
}
```

**输入示例：**
```
a1b2c3
```
**输出示例：**
```
6
```

### 例题3：回文串判断（难度 ★★★）

**题意：** 读入一个字符串，判断它是不是**回文串**（正着读和反着读一样，比如 "abcba"、"noon"）。输出 "Yes" 或 "No"。

**思路讲解：**
> 回文串就像照镜子——左边和右边是对称的。我们可以用两个"指针"，一个从最左边出发，一个从最右边出发，两个一起往中间走，每一步检查它们指向的字符是不是一样。如果有一对不一样，就不是回文串。

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string s;
    cin >> s;
    bool ok = true;    // 先假设是回文
    int left = 0, right = s.size() - 1;
    while(left < right){
        if(s[left] != s[right]){
            ok = false;   // 发现不对称，不是回文
            break;
        }
        left++;
        right--;
    }
    if(ok) cout << "Yes" << endl;
    else cout << "No" << endl;
    return 0;
}
```

**输入示例1：** `abcba` → **输出：** `Yes`

**输入示例2：** `hello` → **输出：** `No`

### 例题4：密码加密（难度 ★★★）

**题意：** 小明设计了一种简单加密方法：把每个小写字母替换为它后面第 3 个字母（如 `'a'→'d'`，`'x'→'a'`，`'y'→'b'`，`'z'→'c'`）。其他字符不变。读入一个字符串，输出加密后的结果。

**思路讲解：**
> 这叫"凯撒密码"，古罗马凯撒大帝就用过！每个字母往后移 3 位。但是 x、y、z 往后移会超出字母表，怎么办？**绕回来**！用取余运算 `%` 来处理。
>
> 公式：`新字符 = (s[i] - 'a' + 3) % 26 + 'a'`

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    string s;
    cin >> s;
    for(int i = 0; i < s.size(); i++){
        if(s[i] >= 'a' && s[i] <= 'z'){
            s[i] = (s[i] - 'a' + 3) % 26 + 'a';
        }
    }
    cout << s << endl;
    return 0;
}
```

**输入示例：** `xyz` → **输出：** `abc`

**输入示例：** `hello` → **输出：** `khoor`

## 七、常见错误与应对

| 序号 | 常见错误 | 典型表现 | 应对方法 |
|------|---------|---------|---------|
| 1 | **`substr` 下标从 1 开始** | 写成 `s.substr(1, 3)` 以为从第 1 个字符开始 | 反复强调"下标从 0 开始"，黑板画火车图，让学生指着数 |
| 2 | **字典序和数值序混淆** | 认为 `"9" < "10"` | 用"查字典"的比喻：只看第一个字母 `'9'` 和 `'1'`，`'9'` 排后面所以更大 |
| 3 | **字符串比较用单等号** | 写成 `if(s1 = s2)` | 提醒：一个等号是"赋值"，两个等号才是"判断相等"——这是老朋友了！ |
| 4 | **大小写转换忘记判断** | 直接写 `s[i] -= 32`，导致非字母字符也被修改 | 强调必须先判断是不是字母，再做转换 |
| 5 | **`find` 返回值直接当 bool** | 写成 `if(s.find("abc"))` | 解释：`find` 返回的是位置，找到位置 0 时条件反而为 false！必须和 `string::npos` 比较 |
| 6 | **循环中 `i < s.size()` 类型警告** | `int i` 和 `size()` 返回的无符号类型比较 | 四年级阶段可以先忽略警告，后续课程再讲；也可写成 `i < (int)s.size()` |
| 7 | **凯撒加密不取余** | `'z' + 3` 变成 `'}'` 而不是 `'c'` | 画一个圆形字母表，演示"到头了就绕回来"的感觉 |
