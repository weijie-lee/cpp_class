## 四、讲稿提纲

### 4.1 导入回顾（8分钟）——"书架不够大怎么办？"

> 同学们，上节课我们学了stack（栈）和queue（队列），大家还记得吧？stack是"弹夹"，后进先出；queue是"排队"，先进先出。它们都是C++工具箱STL里的好东西。
>
> 今天我们要学STL里另一个**超级常用**的工具——vector。它有多常用呢？可以说在CSP-J比赛里，**几乎每场必用**！
>
> 在学vector之前，老师先问大家一个问题。我们之前用数组的时候，比如 `int a[100]`，这个100是写死的对不对？**如果我不知道要存多少个数怎么办呢？**
>
> 打个比方——普通数组就像家里的**固定书架**。你买的时候就3层，最多放30本书。有一天你的书越来越多，放不下了！怎么办？买新书架？搬家？太麻烦了！
>
> 要是有一种**神奇的书架**——你每放一本书，它就自动多出一格空间，永远不会满，那该多好啊！
>
> 这种"神奇的书架"在C++里就是——**vector**！

**（板书：普通数组 = 固定书架🗄️，vector = 可以无限加层的书架📚）**

### 4.2 新课讲解上半场（22分钟）——vector核心操作

#### 第一步：vector怎么创建？（4分钟）

> 要用vector，首先需要引入头文件：

```cpp
#include<vector>
```

> 然后声明一个vector也很简单：

```cpp
vector<int> v;
```

> 我们来看这行代码：
> - `vector` 是类型名，告诉电脑"我要一个动态数组"
> - `<int>` 是尖括号，告诉电脑"里面存整数"——跟第21课学的 `greater<int>()` 那个尖括号是一回事
> - `v` 是你给它起的名字，叫什么都行，叫 `a`、`arr`、`nums` 都可以
>
> 记住：**刚创建出来的vector是空的**，里面一个元素都没有。就像你买了一个全新的、可以伸缩的书架，还没往上面放书。

**（板书：`vector<int> v;` → 创建一个空的、存int的动态数组）**

#### 第二步：往vector里加东西——push_back()（5分钟）

> 书架有了，怎么往里面放书呢？用 `push_back()`：

```cpp
v.push_back(10);   // 放一个10进去
v.push_back(20);   // 再放一个20
v.push_back(30);   // 再放一个30
// 现在v里面是：{10, 20, 30}
```

> `push_back` 的意思是"从**后面**推进去"。还记得上节课stack的 `push` 吗？vector的 `push_back` 多了一个 `_back`，意思是"推到**最后面**"。
>
> 打个比方：你在书架的**最右边**一本一本地放书，先放的在左边，后放的在右边。
>
> 注意写法：是 `v.push_back(10)`，中间有个小圆点`.`。这个小圆点你可以理解为"**的**"——`v` 的 `push_back`，就是"对v这个书架执行push_back操作"。

**（在电脑上演示：每push_back一次，打印一下vector的内容，让学生直观看到变化）**

#### 第三步：查看大小和取走最后一个——size()、pop_back()、empty()（5分钟）

> 我想知道书架上现在有几本书？用 `v.size()`：

```cpp
cout << v.size();  // 输出3，因为我们放了3个元素
```

> `size` 就是"大小、数量"的意思。它会告诉你vector里**现在有几个元素**。
>
> 想把最后一本书取下来？用 `v.pop_back()`：

```cpp
v.pop_back();  // 取走最后一个元素（30被移除了）
// 现在v里面是：{10, 20}
```

> 还记得stack的 `pop` 吗？这里也是一样的意思——弹出去、拿走。`pop_back` 就是"从**最后面**拿走一个"。
>
> 还有一个好用的：`v.empty()`，判断vector是不是空的。空的返回 `true`，不空返回 `false`。

```cpp
if(v.empty()){
    cout << "书架是空的！";
}
```

**（板书汇总表：）**

| 操作 | 写法 | 意思 | 比喻 |
|------|------|------|------|
| 加元素 | `v.push_back(x)` | 在最后面加一个x | 书架最右边放一本书 |
| 删元素 | `v.pop_back()` | 删掉最后一个 | 拿走最右边那本书 |
| 看大小 | `v.size()` | 现在有几个元素 | 书架上有几本书 |
| 是否为空 | `v.empty()` | 是不是空的 | 书架上有没有书 |
| 访问元素 | `v[i]` | 取第i个元素 | 拿第i格的那本书 |

#### 第四步：访问和遍历——v[i] 和for循环（8分钟）

> 我想拿vector里第2个元素怎么办？跟数组一样，用中括号：

```cpp
cout << v[0];  // 输出第1个元素
cout << v[1];  // 输出第2个元素
```

> **跟数组一模一样，下标也是从0开始的！** 这一点大家应该很熟悉了。
>
> 那怎么把vector里的所有元素都输出来呢？用for循环遍历：

```cpp
for(int i = 0; i < v.size(); i++){
    cout << v[i] << " ";
}
```

> 注意看：循环条件是 `i < v.size()`，不是写一个固定的数字，而是用 `v.size()` 来获取元素个数。因为vector的大小是会变的，所以我们要**动态获取**它的大小。
>
> 同学们，到这里vector的核心操作就这些了。我们来总结一下vector和普通数组的对比：

**（板书对比表）**

| | 普通数组 `int a[100]` | vector `vector<int> v` |
|---|---|---|
| 大小 | 固定不变 | 随时可以变大变小 |
| 加元素 | 不能加，满了就溢出 | `push_back()` 随便加 |
| 知道有几个 | 自己用变量记 | `v.size()` 自动知道 |
| 访问方式 | `a[i]` | `v[i]`（一样的！） |
| 适合场景 | 知道最多有多少个 | 不确定有多少个 |

> **竞赛建议：** 如果题目告诉你数据最多有多少个，用普通数组就够了；如果不确定，或者需要动态增删，就用vector。后面学图论的时候，vector会特别特别好用！

### 4.3 新课讲解下半场（18分钟）——set、map简介 + 万能头文件

#### 第五步：认识set——不重复的集合（6分钟）

> 同学们，vector学会了，接下来老师带大家**认识**两个新朋友。注意是**认识**，不是要你们精通，今天只要知道它们能干什么就行。
>
> 第一个叫 **set**（集合）。
>
> 打个比方：班级要选课，老师在黑板上写已经报名的同学名字。如果张三已经在上面了，你再写一次"张三"，老师会说"**已经有了，不用重复写**"。
>
> set就是这样一个"不允许重复"的容器。你往里面丢数字，它会**自动去重**，还会**自动排好序**。

```cpp
#include<set>
using namespace std;

set<int> s;
s.insert(3);  // 放入3
s.insert(1);  // 放入1
s.insert(3);  // 再放3——但是不会重复放！
s.insert(2);  // 放入2
// 此时s里面是：{1, 2, 3}，自动去重、自动排序
cout << s.size(); // 输出3，不是4！
```

> set什么时候有用呢？比如题目问你"一共出现了多少个不同的数"，用set就特别方便——全部塞进去，最后 `s.size()` 就是答案。

#### 第六步：认识map——键值对字典（6分钟）

> 第二个新朋友叫 **map**（映射），你可以把它理解为一本**字典**。
>
> 字典是什么？你查一个**单词**（键），就能找到它的**解释**（值）。map就是这样：给它一个"键"，它对应一个"值"。
>
> 比如我们想记录每个同学的考试分数：

```cpp
#include<map>
using namespace std;

map<string, int> score;
score["小明"] = 95;
score["小红"] = 88;
score["小刚"] = 100;
cout << score["小红"];  // 输出88
```

> 是不是很像查字典？"小红的分数是多少？"——88！
>
> map还有一个常见用法：**统计每个数出现了几次**。

```cpp
map<int, int> cnt;
cnt[5]++;  // 数字5出现了1次
cnt[3]++;  // 数字3出现了1次
cnt[5]++;  // 数字5又出现了1次
cout << cnt[5]; // 输出2，5出现了2次
```

> set和map今天只需要**知道有这么个东西**，以后遇到相关题目我们会再详细讲。

#### 第七步：万能头文件——竞赛神器（3分钟）

> 同学们有没有觉得，学了这么多STL，头文件好多啊？用sort要写 `#include<algorithm>`，用stack要写 `#include<stack>`，用vector要写 `#include<vector>`，用set要写 `#include<set>`……记不住怎么办？
>
> 老师教你们一个**竞赛神器**——万能头文件：

```cpp

```

> 这一行顶替所有头文件！写了它，你就不需要再写 `<iostream>`、`<algorithm>`、`<vector>`、`<stack>`、`<queue>`、`<set>`、`<map>`……全都包含了！
>
> 你可以把它想象成：以前你去上学，要自己一样一样带——铅笔、橡皮、尺子、书本。现在有一个**万能书包**，什么都装好了，背上就走！
>
> **注意：** 万能头文件只能在竞赛和平时练习中用。正式的软件开发项目中不推荐用，因为它会拖慢编译速度。但在竞赛里，**强烈推荐**！

**（板书：`#include<bits/stdc++.h>` ← 万能头文件，竞赛必备！）**

## 五、课堂例题

### 例题1：动态记录成绩（vector基础入门）

**题目：** 老师一个一个念学生成绩，你不知道一共有多少个学生。输入若干个正整数表示成绩，以0作为结束标志（0不算成绩）。输出所有成绩和学生人数。

**输入样例：**
```
88 95 72 100 65 0
```

**输出样例：**
```
88 95 72 100 65
5
```

**完整代码：**

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    vector<int> v;
    int x;
    while(cin >> x){
        if(x == 0) break;  // 遇到0就停止
        v.push_back(x);    // 把成绩放进vector
    }
    // 遍历输出所有成绩
    for(int i = 0; i < v.size(); i++){
        if(i > 0) cout << " ";
        cout << v[i];
    }
    cout << endl;
    cout << v.size() << endl;  // 输出学生人数
    return 0;
}
```

**讲解思路：**

> 这道题的关键在于——我们**不知道**一共会输入多少个数字。如果用普通数组，你得猜一个大小比如 `int a[10000]`，万一猜少了就炸了。
>
> 用vector就完美了！来一个，`push_back` 一个，它自己会变大。最后用 `v.size()` 就知道一共收了多少个。
>
> 注意我们用了万能头文件 `#include<bits/stdc++.h>`，所以不需要单独写 `#include<vector>`。以后大家竞赛代码就这么写就行！

### 例题2：vector排序与查找最值

**题目：** 输入n个整数，存入vector中，然后：①从小到大排序后输出；②输出最大值和最小值。

**输入样例：**
```
5
3 1 4 1 5
```

**输出样例：**
```
1 1 3 4 5
max=5 min=1
```

**完整代码：**

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n;
    cin >> n;
    vector<int> v(n);  // 创建一个大小为n的vector
    for(int i = 0; i < n; i++){
        cin >> v[i];
    }
    // sort也能用在vector上！
    sort(v.begin(), v.end());
    for(int i = 0; i < v.size(); i++){
        if(i > 0) cout << " ";
        cout << v[i];
    }
    cout << endl;
    // 排序后，最小值在最前面，最大值在最后面
    cout << "max=" << v[v.size()-1] << " min=" << v[0] << endl;
    return 0;
}
```

**讲解思路：**

> 这道题有两个新知识点：
>
> **第一个：** `vector<int> v(n)` ——创建vector的时候直接指定大小为n，这样就可以用 `v[i]` 来输入了。注意是**圆括号**里写n，不是方括号。
>
> **第二个：** sort可以对vector排序！但写法跟数组有点不同：
> - 数组排序：`sort(a, a+n)`
> - vector排序：`sort(v.begin(), v.end())`
>
> `v.begin()` 是vector的"开头"，`v.end()` 是vector的"末尾后面一位"。你可以把它们理解为"从头到尾排一遍"。
>
> 排好序之后，最小值在 `v[0]`，最大值在 `v[v.size()-1]`，也就是最后一个。

### 例题3：统计不同数字的个数（set应用）

**题目：** 输入n个整数，问一共有多少个**不同的**数字？

**输入样例：**
```
7
1 2 3 2 1 4 2
```

**输出样例：**
```
4
```

**完整代码：**

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n;
    cin >> n;
    set<int> s;
    for(int i = 0; i < n; i++){
        int x;
        cin >> x;
        s.insert(x);  // 重复的会自动忽略
    }
    cout << s.size() << endl;
    return 0;
}
```

**讲解思路：**

> 如果不用set，你要自己去重——用循环一个个比较，看这个数有没有出现过，代码很长。
>
> 用set就太简单了：全部 `insert` 进去，它**自动帮你去掉重复的**，最后 `s.size()` 就是不同数字的个数。
>
> 1、2、3、4 四个不同的数字，虽然输入了7个数，但2出现了3次、1出现了2次，set里每个只保留一份。
>
> 这就是set的魅力——去重神器！大家记住这个用法就好。

### 例题4：统计每个数字出现次数（map应用）

**题目：** 输入n个整数（范围1~100），输出每个出现过的数字及其出现次数，按数字从小到大的顺序输出。

**输入样例：**
```
8
3 1 2 3 1 3 2 1
```

**输出样例：**
```
1 3
2 2
3 3
```

**完整代码：**

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n;
    cin >> n;
    map<int, int> cnt;
    for(int i = 0; i < n; i++){
        int x;
        cin >> x;
        cnt[x]++;  // x出现的次数加1
    }
    // map会自动按键(key)从小到大排序
    for(auto it = cnt.begin(); it != cnt.end(); it++){
        cout << it->first << " " << it->second << endl;
    }
    return 0;
}
```

**讲解思路：**

> 这道题展示了map的经典用法——**统计频率**（每个数出现了几次）。
>
> `cnt[x]++` 这行代码太妙了：如果x第一次出现，`cnt[x]` 从0开始，加1变成1；再出现就加到2、3……
>
> 遍历map的写法看起来比较长，大家暂时不用死记硬背，知道有这种用法就行。后面遇到类似题目老师会再讲。
>
> 另外告诉大家：如果数据范围小（比如1~100），其实用普通数组 `int cnt[101]` 也能统计频率，速度更快。map的优势在于**数据范围很大或者键不是整数**的时候。

## 六、课堂练习题

### 练习题1（基础）：倒序输出

**题目：** 输入若干个正整数，以-1作为结束标志（-1不算数据）。请将这些数**倒序**输出（最后输入的最先输出）。

**输入样例：**
```
5 3 8 2 7 -1
```

**输出样例：**
```
7 2 8 3 5
```

**提示：** 用vector存储所有数据，然后从最后一个元素开始倒着遍历输出（`for(int i = v.size()-1; i >= 0; i--)`）。

## 七、常见错误与应对

### 错误1：忘记写头文件或写错头文件

```
error: 'vector' was not declared in this scope
```

**应对：** 最简单的办法——**直接用万能头文件** `#include<bits/stdc++.h>`，一劳永逸。建议学生从今天开始，竞赛代码统一使用万能头文件。

---

### 错误2：vector声明时用了方括号

```cpp
vector<int> v[10];   // 错误理解！这是创建了10个vector，不是一个大小为10的vector！
vector<int> v(10);   // 正确！创建大小为10、初始值为0的vector
```

**应对：** 强调：圆括号 `()` 设置大小，方括号 `[]` 在vector前面是创建"vector数组"（多个vector），含义完全不同。初学者统一用 `vector<int> v;` 加 `push_back()` 来添加元素最安全。

---

### 错误3：对空vector使用v[0]访问

```cpp
vector<int> v;
cout << v[0];  // 危险！v是空的，没有第0个元素！
```

**应对：** 用 `v[i]` 访问之前，确保vector里有足够的元素。可以先用 `v.size()` 检查，或者确保已经 `push_back` 过。空vector直接用下标访问不会报错但结果不可预料（未定义行为），这是一个隐蔽的bug。

---

### 错误4：sort对vector排序时写成数组的写法

```cpp
sort(v, v + n);           // 错误！vector不能这样写
sort(v.begin(), v.end()); // 正确！
```

**应对：** 提醒学生区分：
- 数组排序：`sort(a, a+n)`
- vector排序：`sort(v.begin(), v.end())`

口诀：**"数组用a，vector用begin/end"**。

---

### 错误5：`v.size()-1` 在vector为空时溢出

```cpp
vector<int> v;  // 空vector
for(int i = 0; i < v.size() - 1; i++){  // 危险！
    // v.size()返回0，0-1溢出成一个超大的数
}
```

**应对：** `v.size()` 返回的是无符号整数（`unsigned`），0减1不会变成-1，而是变成一个非常大的数，导致循环出错。解决方法：先判断 `if(v.size() > 0)` 再做减法，或者将 `v.size()` 强制转换为 `int`。
