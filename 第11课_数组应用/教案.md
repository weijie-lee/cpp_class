## 四、讲稿提纲

### 4.1 热身回顾（5 min）

> 同学们，上节课我们学了一个超级厉害的东西——**数组**！还记得吗？数组就像一排**储物柜**，每个柜子有自己的编号，从 0 号开始。我们可以用 `for` 循环一个一个地打开柜子放东西、拿东西。
>
> 快速抢答：
> - `int a[5];` 创建了几个柜子？编号是什么？（5 个，0~4）
> - 怎么把第 3 个柜子的值变成 100？（`a[2] = 100;`）
> - 怎么把所有柜子里的数打印出来？（`for` 循环遍历）

### 4.2 讲解①：擂台法求最大值 / 最小值（15 min）

#### 故事引入

> 同学们看过武林大会吗？擂台赛是这样的：**第一个选手先上台当擂主**，然后一个一个挑战者上来比武。如果挑战者赢了，挑战者就变成新擂主；如果挑战者输了，擂主不变。所有人比完以后，**最后站在台上的那个人就是冠军**！
>
> 找数组里的最大值，用的就是一模一样的方法！我们叫它**"打擂台法"**。
>
> - 擂台（变量 `maxVal`）：一开始让 `a[0]` 当擂主
> - 挑战者：`a[1]`、`a[2]`……一个一个来
> - 比较规则：如果挑战者 > 擂主，挑战者上台（更新 `maxVal`）
> - 最后 `maxVal` 就是最大值！

#### 板书/PPT 演示

用具体数字演示过程：`a[] = {3, 7, 2, 9, 5}`

| 回合 | 擂主(maxVal) | 挑战者 | 比较结果 | 新擂主 |
|------|-------------|--------|---------|--------|
| 初始 | 3 (a[0]) | — | — | 3 |
| 第 1 轮 | 3 | 7 (a[1]) | 7 > 3 ✔ | **7** |
| 第 2 轮 | 7 | 2 (a[2]) | 2 > 7 ✘ | 7 |
| 第 3 轮 | 7 | 9 (a[3]) | 9 > 7 ✔ | **9** |
| 第 4 轮 | 9 | 5 (a[4]) | 5 > 9 ✘ | 9 |

> 最终擂主是 **9**，就是最大值！

#### 易错点强调

> ⚠️ 擂主一开始应该让谁当？**让 `a[0]` 当**，这是最安全的！
> 有些同学可能想"让 0 当擂主"——如果数组里都是负数呢？`-3, -7, -1`，0 比它们都大，结果就错了！
> 所以记住口诀：**"擂主起手 a 零号"**。

### 4.3 讲解②：求平均值 + 条件统计（20 min）

#### 平均值——先加后除

> 求平均分，就像把全班同学的糖果都**倒进一个大桶**（累加求和），然后**平均分给每个人**（除以人数）。
>
> 两个关键：
> 1. **大桶**（变量 `sum`）要从 0 开始
> 2. **除法要小心**！`10 / 3` 在 C++ 里等于 `3` 不是 `3.33`！我们需要写 `1.0 * sum / n` 才能得到带小数的结果。

#### 条件统计——数数

> 统计及格人数，就像老师点名："**成绩大于等于 60 的同学请举手！**"我们数一数举手的人数就行了。
>
> 在程序里，我们用一个**计数器** `count = 0`，遍历数组，每遇到一个 `a[i] >= 60` 就 `count++`。

#### 整除 vs 浮点除法——现场实验

> 我们来做一个小实验！在电脑上运行：
> ```
> cout << 10 / 3 << endl;      // 输出 3（整数除法，小数丢了！）
> cout << 10.0 / 3 << endl;    // 输出 3.33333（浮点除法）
> cout << 1.0 * 10 / 3 << endl; // 输出 3.33333（乘 1.0 变浮点）
> ```
> 记住：**想要小数点，就乘个 1.0！**

### 4.4 讲解③：数组查找 + 综合应用（20 min）

#### 数组查找

> 在一排储物柜里找某个东西，只能**一个一个打开看**。这就是**顺序查找**。
>
> 遍历数组，如果 `a[i] == x`，就找到了！可以输出位置 `i`，或者用 `flag` 标记表示"找到了"。

#### 综合应用：成绩分析

> 现在我们把前面学的所有招式**合体**！老师输入全班 N 个同学的成绩，**一次遍历**就能算出：最高分、最低分、平均分、及格人数。
>
> 这就像流水线工厂，一个产品（数据）经过的时候，**四个检查员同时工作**：
> - 1 号检查员：跟当前最大值比
> - 2 号检查员：跟当前最小值比
> - 3 号检查员：加到总分里
> - 4 号检查员：看看是否及格，及格就计数

## 五、课堂例题

### 例题 1：找最大值（擂台法入门）

**题目**：输入 n 个整数，找出其中的最大值并输出。

**输入样例**：
```
5
3 7 2 9 5
```
**输出样例**：
```
9
```

**完整代码**：

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[1005];  // 开大一点，养成好习惯
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    // 打擂台：让 a[0] 先当擂主
    int maxVal = a[0];
    for (int i = 1; i < n; i++) {   // 注意从 1 开始！a[0] 已经是擂主了
        if (a[i] > maxVal) {
            maxVal = a[i];  // 挑战者赢了，换擂主
        }
    }

    cout << maxVal << endl;
    return 0;
}
```

**讲解思路**：
1. 先读入数据存到数组里——这是上节课学过的。
2. 让 `a[0]` 当擂主（`maxVal = a[0]`）。
3. 从 `a[1]` 开始挑战，`i` 从 `1` 开始循环。
4. 如果 `a[i] > maxVal`，说明挑战者更强，擂主换人。
5. 循环结束，`maxVal` 就是冠军！

> 💡 **思考题**：如果要找最小值呢？只要把擂台规则改一下——"谁更小谁上台"，把 `>` 改成 `<` 就行了！

### 例题 2：求平均值

**题目**：输入 n 个整数，求它们的平均值（保留 2 位小数）。

**输入样例**：
```
4
80 90 70 85
```
**输出样例**：
```
81.25
```

**完整代码**：

```cpp
#include <iostream>
#include <iomanip>   // 用于 fixed 和 setprecision
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[1005];
    int sum = 0;  // 大桶，一开始是空的

    for (int i = 0; i < n; i++) {
        cin >> a[i];
        sum += a[i];  // 边读边加，倒进大桶
    }

    // 注意：1.0 * sum 把整数变成小数再除
    double avg = 1.0 * sum / n;
    cout << fixed << setprecision(2) << avg << endl;

    return 0;
}
```

**讲解思路**：
1. `sum` 初始化为 0——大桶一开始是空的。
2. 读入的同时就累加，不需要单独再遍历一次（当然分开写也可以，但合在一起更简洁）。
3. **重点**：`1.0 * sum / n`，先乘 `1.0` 让 `sum` 变成浮点数，再除以 `n`，这样结果才有小数。
4. `fixed << setprecision(2)` 控制输出保留 2 位小数。

> 💡 **易错警示**：如果写 `sum / n`，`80+90+70+85=325`，`325 / 4 = 81`（整数除法！），小数部分被丢掉了！

### 例题 3：统计及格人数

**题目**：输入 n 个同学的成绩（0~100 的整数），统计及格（≥60 分）的人数。

**输入样例**：
```
6
55 80 42 60 99 38
```
**输出样例**：
```
3
```

**完整代码**：

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[1005];
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    int count = 0;  // 计数器，一开始是 0
    for (int i = 0; i < n; i++) {
        if (a[i] >= 60) {
            count++;  // 发现一个及格的，数一下
        }
    }

    cout << count << endl;
    return 0;
}
```

**讲解思路**：
1. `count = 0`——还没开始数，当然是 0。
2. 遍历数组，每个元素都问一句："你及格了吗？（`>= 60`？）"
3. 及格就 `count++`，不及格跳过。
4. 最后 `count` 就是及格人数。

> 💡 **拓展提问**：如果要统计"优秀（≥90 分）"的人数呢？只要把条件 `>= 60` 改成 `>= 90` 就行了！

### 例题 4：综合应用——成绩分析（重点！）

**题目**：输入 n 个同学的成绩，求最高分、最低分、平均分（保留 2 位小数）和及格人数。

**输入样例**：
```
5
88 55 92 60 73
```
**输出样例**：
```
max=92
min=55
avg=73.60
pass=3
```

**完整代码**：

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[1005];
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    // 四个"检查员"同时上岗
    int maxVal = a[0];   // 1 号：最高分擂主
    int minVal = a[0];   // 2 号：最低分擂主
    int sum = 0;          // 3 号：分数大桶
    int count = 0;        // 4 号：及格计数器

    for (int i = 0; i < n; i++) {
        // 1 号检查员：打擂台找最大
        if (a[i] > maxVal) {
            maxVal = a[i];
        }
        // 2 号检查员：打擂台找最小
        if (a[i] < minVal) {
            minVal = a[i];
        }
        // 3 号检查员：倒进大桶
        sum += a[i];
        // 4 号检查员：数及格人数
        if (a[i] >= 60) {
            count++;
        }
    }

    double avg = 1.0 * sum / n;

    cout << "max=" << maxVal << endl;
    cout << "min=" << minVal << endl;
    cout << "avg=" << fixed << setprecision(2) << avg << endl;
    cout << "pass=" << count << endl;

    return 0;
}
```

**讲解思路**：
1. 这道题是前面三道题的"合体版"！我们同时完成四项任务。
2. **关键思想**：只需要**一次遍历**，四个变量各干各的活——这就是编程的高效！
3. 注意 `sum` 是从 `0` 开始累加，所以循环从 `i = 0` 开始；而 `maxVal` 和 `minVal` 已经用 `a[0]` 初始化了，从 `i = 0` 开始也没问题（第一轮 `a[0]` 和自己比，不影响结果）。
4. 最后分别输出四个结果。

> 🏆 **成就感时刻**：同学们，你们刚才写的这个程序，其实就是老师每次考完试用来统计成绩的工具！你们已经能写"老师用的程序"了，是不是很厉害？

## 七、常见错误与应对

### 错误 1：擂主初始值设为 0

```cpp
int maxVal = 0;  // ❌ 错误！
```

**问题**：如果数组里都是负数，例如 `{-3, -7, -1}`，`maxVal` 始终是 0，但 0 并不在数组里！

**正确做法**：
```cpp
int maxVal = a[0];  // ✅ 让数组第一个元素当擂主
```

**口诀**：**"擂主起手 a 零号，不用 0 来瞎胡闹"**。

---

### 错误 2：整数除法丢小数

```cpp
double avg = sum / n;  // ❌ 整数除整数，小数部分丢了！
```

**问题**：`sum` 和 `n` 都是 `int`，`325 / 4 = 81`，赋值给 `double` 变成 `81.0`，不是 `81.25`。

**正确做法**：
```cpp
double avg = 1.0 * sum / n;  // ✅ 先变浮点再除
```

**口诀**：**"求平均，别忘乘个一点零"**。

---

### 错误 3：计数器忘记初始化

```cpp
int count;  // ❌ 没给初始值！
count++;    // 从一个不确定的数开始加，结果不对
```

**正确做法**：
```cpp
int count = 0;  // ✅ 从 0 开始数
```

**口诀**：**"计数器，先归零"**。

---

### 错误 4：数组越界

```cpp
int a[5];
for (int i = 0; i <= 5; i++) {  // ❌ i=5 时越界！a[5] 不存在！
    cin >> a[i];
}
```

**正确做法**：
```cpp
for (int i = 0; i < 5; i++) {  // ✅ i 从 0 到 4
```

**口诀**：**"数组五个 0 到 4，等号千万不要带"**。

---

### 错误 5：求最大值时循环从 0 开始但忘了初始化

```cpp
int maxVal;  // ❌ 没初始化！
for (int i = 0; i < n; i++) {
    if (a[i] > maxVal) {  // maxVal 是个随机数，比较无意义
        maxVal = a[i];
    }
}
```

**正确做法**：要么 `maxVal = a[0]` 从 `i = 1` 开始循环，要么 `maxVal = a[0]` 从 `i = 0` 开始循环（多比一次但不出错）。
