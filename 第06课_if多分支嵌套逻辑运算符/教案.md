### 4.2 讲解 1：if - else if - else 多分支（20 分钟）

#### 🎯 生活引入

> 上节课的岔路口只有两条路。但生活中，很多路口有**好几条路**——
>
> 想象你走进一家奶茶店，菜单上写着：
> - 大杯 → 12 元
> - 中杯 → 9 元
> - 小杯 → 6 元
>
> 服务员问你要什么杯型，然后告诉你价格。这就不是"二选一"了，而是"**多选一**"。
>
> 在 C++ 里，我们用 **`if - else if - else`** 来处理这种"多条岔路"的情况。

#### 📖 语法讲解

```
if (条件1) {
    // 条件1 满足时执行
} else if (条件2) {
    // 条件1 不满足、条件2 满足时执行
} else if (条件3) {
    // 前面都不满足、条件3 满足时执行
} else {
    // 以上都不满足时执行
}
```

> **关键规则：从上到下，谁先满足就走谁，走了一个就跳过后面所有的。**
>
> 就像排队检票——第一个闸机能过就过，过了就不用看后面的闸机了。

#### 🔑 强调要点

- `else if` 可以写很多个，`else` 最多一个，也可以不写。
- 顺序很重要！条件的范围要从严到宽（或从高到低），否则会被"拦截"。
- 每个分支的 `{}` 不要忘！

---

### 4.3 讲解 2：逻辑运算符（15 分钟）

#### 🎯 生活引入

> 妈妈说："**作业写完了，而且房间收拾好了**，才能玩游戏。"
> 这里有两个条件，**两个都要满足**才行——这就是 `&&`（而且）。
>
> 爸爸说："**周六或者周日**，可以晚起床。"
> 两个条件，**满足一个就行**——这就是 `||`（或者）。
>
> 老师说："**没有**迟到的同学，奖一颗星。"
> 把一个条件"**反过来**"——这就是 `!`（不是 / 取反）。

#### 📖 三个运算符

| 运算符 | 读法 | 含义 | 生活比喻 |
|--------|------|------|----------|
| `&&` | 而且（AND） | 两边**都为真**，结果才为真 | 钥匙 + 密码都对才能开保险箱 |
| `\|\|` | 或者（OR） | 两边**任一为真**，结果就为真 | 有学生证或工作证都能买票 |
| `!` | 不是（NOT） | 把真变假、假变真 | "不下雨"就是把"下雨"反过来 |

#### 🎲 真值表小游戏（互动）

> 老师说两个条件，同学们举手回答"能不能玩游戏"：
>
> | 作业写完？ | 房间整理？ | 能玩吗（&&）？ |
> |:---:|:---:|:---:|
> | ✅ | ✅ | ✅ 能！ |
> | ✅ | ❌ | ❌ 不能 |
> | ❌ | ✅ | ❌ 不能 |
> | ❌ | ❌ | ❌ 不能 |
>
> 换成 `||`（满足一个就行）呢？再来一遍！

#### 🔑 强调要点

- `&&` 优先级高于 `||`，混合用时**加括号**最安全。
- `!` 作用于紧跟其后的一个条件，多个条件要用 `!(a && b)` 这样包起来。

### 4.4 讲解 3：if 嵌套（15 分钟）

#### 🎯 生活引入

> 想象你去游乐园坐过山车：
> - **第一道门**：检查身高——够 1.2 米才能进。
> - 进去以后还有**第二道门**：检查年龄——10 岁以上坐 A 车，10 岁以下坐 B 车。
>
> 这就是"**条件里面还有条件**"——if 嵌套！

#### 📖 语法讲解

```cpp
if (身高 >= 120) {
    // 过了第一道门
    if (年龄 >= 10) {
        cout << "坐A车" << endl;
    } else {
        cout << "坐B车" << endl;
    }
} else {
    cout << "身高不够，不能坐过山车" << endl;
}
```

> **内层的 if-else 整个都装在外层 if 的大括号里面。**
>
> 写的时候，记住"**每进一层，往右缩进一次**"，代码就不会乱。

#### 🔑 强调要点

- 嵌套一般不超过两到三层，太深了自己都会晕。
- 很多时候嵌套可以用 `&&` 改写成一句，要学会灵活选择。

> 比如上面的代码也可以写成：
> ```cpp
> if (身高 >= 120 && 年龄 >= 10) {
>     cout << "坐A车" << endl;
> }
> ```
> 但如果不同情况要做不同的事，嵌套就更清晰。

## 五、课堂例题

### 例题 1：成绩等级判断（多分支入门）

**题目**：输入一个 0–100 的整数成绩，输出等级：
- 90 分及以上：`A`
- 80–89 分：`B`
- 60–79 分：`C`
- 60 分以下：`D`

**讲解思路**：
1. 先想：这里有 4 种情况，用 `if - else if - else if - else` 刚好 4 条路。
2. 注意顺序：从高到低判断，因为 `else if` 已经隐含"前面不满足"的条件。
3. 写完后拿 95、85、70、50 四个数验证。

```cpp
#include <iostream>
using namespace std;

int main() {
    int score;
    cin >> score;

    if (score >= 90) {
        cout << "A" << endl;
    } else if (score >= 80) {
        cout << "B" << endl;
    } else if (score >= 60) {
        cout << "C" << endl;
    } else {
        cout << "D" << endl;
    }

    return 0;
}
```

> 💡 **带着学生模拟**：输入 75，走到第一个 `if`——75 >= 90？不满足，跳过。走到第二个——75 >= 80？不满足，跳过。走到第三个——75 >= 60？满足！输出 C，后面的 else 全部跳过。

### 例题 2：简单计算器（多分支进阶）

**题目**：输入两个整数 a、b 和一个运算符（`+`、`-`、`*`），输出运算结果。如果运算符不是这三种之一，输出 `Error`。

**讲解思路**：
1. 运算符是一个字符，用 `char` 存。
2. 四条路：`+`、`-`、`*`、其他。
3. 注意字符比较用单引号 `'+'`。

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    char op;
    cin >> a >> op >> b;

    if (op == '+') {
        cout << a + b << endl;
    } else if (op == '-') {
        cout << a - b << endl;
    } else if (op == '*') {
        cout << a * b << endl;
    } else {
        cout << "Error" << endl;
    }

    return 0;
}
```

> 💡 **拓展提问**：如果要加上除法 `/`，需要注意什么？（引导学生说出"除数不能为 0"——这就是嵌套的应用场景！）

---

### 例题 3：闰年判断（逻辑运算符综合）

**题目**：输入一个年份，判断是不是闰年。

**规则**：
- 能被 4 整除**且**不能被 100 整除 → 闰年
- 能被 400 整除 → 也是闰年
- 其他 → 平年

**讲解思路**：
1. 这道题有"而且"（&&）和"或者"（||），是逻辑运算符的经典题。
2. 翻译成代码：`(year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)`
3. 用 2024、1900、2000、2023 四个年份验证。

```cpp
#include <iostream>
using namespace std;

int main() {
    int year;
    cin >> year;

    if ((year % 4 == 0 && year % 100 != 0) || year % 400 == 0) {
        cout << "是闰年" << endl;
    } else {
        cout << "不是闰年" << endl;
    }

    return 0;
}
```

> 💡 **手动模拟**：
> - 输入 2024：2024 % 4 == 0 ✅ 且 2024 % 100 != 0 ✅ → 前半段为真 → 整体为真 → 闰年 ✅
> - 输入 1900：1900 % 4 == 0 ✅ 但 1900 % 100 != 0 ❌ → 前半段为假；1900 % 400 == 0？❌ → 后半段也为假 → 整体为假 → 平年 ✅
> - 输入 2000：前半段为假（2000 % 100 == 0）；但 2000 % 400 == 0 ✅ → 后半段为真 → 整体为真 → 闰年 ✅

### 例题 4：游乐园门票（if 嵌套）

**题目**：游乐园规则如下：
- 身高不足 120 cm，不能入园。
- 身高 120 cm 及以上可以入园：
  - 年龄 < 12 岁，票价 50 元
  - 年龄 >= 12 岁，票价 100 元

输入身高 h 和年龄 age，输出对应信息。

**讲解思路**：
1. 第一层 if：判断身高够不够。
2. 第二层 if：在身高够的前提下，再判断年龄。
3. 强调缩进——每嵌套一层，多缩进 4 个空格。

```cpp
#include <iostream>
using namespace std;

int main() {
    int h, age;
    cin >> h >> age;

    if (h >= 120) {
        if (age < 12) {
            cout << "欢迎入园！票价50元" << endl;
        } else {
            cout << "欢迎入园！票价100元" << endl;
        }
    } else {
        cout << "身高不足，暂时不能入园哦" << endl;
    }

    return 0;
}
```

> 💡 **思考题**：这道题能不能不嵌套，用 `&&` 改写？（可以！引导学生改写并对比。）

- 18 ≤ BMI < 24：`正常`
- 24 ≤ BMI < 28：`偏胖`
- BMI ≥ 28：`肥胖`

> **考察**：`if - else if - else` 多分支 + `double` 类型运算。

### 练习 2：能不能出去玩？（逻辑运算符）

输入三个整数：
- `homework`（0 表示没写完，1 表示写完了）
- `rain`（0 表示没下雨，1 表示下雨了）
- `weekend`（0 表示工作日，1 表示周末）

规则：**作业写完了，而且（没下雨 或者 是周末）**，才能出去玩。否则输出"不能出去玩"。

> **考察**：`&&` 与 `||` 的组合使用，括号的运用。

### 练习 3：快递费计算（多分支 + 嵌套）

- 省内件：重量 ≤ 1kg 收 8 元，每超 1kg 加 2 元
- 省外件：重量 ≤ 1kg 收 12 元，每超 1kg 加 5 元

输入类型 type（1 表示省内，2 表示省外）和重量 w（整数，单位 kg），输出快递费。

> **考察**：先判断省内/省外（外层 if），再根据重量计算（内层 if 或直接计算）——综合运用嵌套。

## 七、常见错误与应对

### ❌ 错误 1：`else if` 写成 `else  if`（多个空格）或 `elseif`（连写）

```cpp
// 错误写法
elseif (score >= 80) { ... }   // ❌ C++ 没有 elseif 关键字
```

> **应对**：告诉学生 `else if` 是两个单词，中间恰好一个空格。可以让他们念出来："else——空格——if"。

---

### ❌ 错误 2：多分支顺序写反，导致条件被"拦截"

```cpp
// 错误：先判断 >= 60，score=95 也会走这条路！
if (score >= 60) {
    cout << "C" << endl;
} else if (score >= 80) {    // 永远到不了这里！
    cout << "B" << endl;
}
```

> **应对**：用"排队检票"比喻——条件宽松的放后面，严格的放前面。带学生用 95 分手动走一遍。

---

### ❌ 错误 3：`&&` 写成 `&`，`||` 写成 `|`

> **应对**：强调"逻辑运算符是**双符号**"。编个口诀：**"而且两根棍（&&），或者两根线（||）"**。

---

### ❌ 错误 4：嵌套 if 时大括号不配对

```cpp
if (h >= 120) {
    if (age < 12) {
        cout << "50元" << endl;
    // 少了一个 }
} else {
    cout << "不能入园" << endl;
}
```

> **应对**：教"写完 `{` 立刻写 `}`，再回来填内容"的习惯。同时利用编辑器的自动缩进检查层级。

---

### ❌ 错误 5：`=` 和 `==` 混淆（延续第5课问题）

```cpp
if (op = '+') { ... }   // ❌ 这是赋值，不是比较！
```

> **应对**：继续强调——"一个等号是**给**，两个等号是**问**"。建议学生读代码时把 `==` 读成"等于吗"。
