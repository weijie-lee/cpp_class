## 四、讲稿提纲

### 4.1 导入故事（5 分钟）

> 同学们，假如老师让你帮忙记录全班 40 个同学的数学成绩。
> 如果用我们之前学的变量，你得写：
> ```
> int a, b, c, d, e, f, g ...
> ```
> 一口气写 40 个变量名，而且还得记住谁是谁——这也太累了吧？🤯
>
> 想象一下学校走廊里的**储物柜**——一排整整齐齐的柜子，每个柜子上面贴着编号 0、1、2、3……
> 你只要说"请打开 5 号柜"，就能拿到里面的东西。
>
> 在 C++ 里，**数组**就是这样一排储物柜！今天我们就来学习它。

### 4.2 数组的概念（讲解①前半）

**核心比喻——储物柜：**

- **一排储物柜** = 一个数组
- **柜子的总数** = 数组的大小
- **柜子的编号** = 下标（index）
- **柜子里放的东西** = 数组元素的值

> 🔑 关键：C++ 的柜子编号是从 **0** 开始的！
> 如果有 5 个柜子，编号是 0、1、2、3、4，**没有 5 号柜**。

画图演示：

```
储物柜：  [ 88 ][ 95 ][ 73 ][ 60 ][ 100 ]
编号：       0      1      2      3       4
```

### 4.3 声明语法（讲解①后半）

```cpp
int a[5];          // 造一排 5 个格子的整数储物柜
```

讲解要点：
- `int` → 每个格子里放整数
- `a` → 这排储物柜的名字
- `[5]` → 一共 5 个格子
- **方括号里必须是常量**（目前阶段告诉学生写一个确定的数字即可）

> 🎯 类比加深：`int a[5];` 就像对学校说："请给我造一排叫 a 的储物柜，一共 5 个格子，每个格子里放整数。"

### 4.4 赋值和访问（讲解①继续）

```cpp
a[0] = 88;    // 把 88 放进 0 号柜
a[1] = 95;    // 把 95 放进 1 号柜
cout << a[0]; // 打开 0 号柜，看看里面是什么 → 输出 88
```

提问互动：
- "如果我想把 100 放进第 3 个格子，应该写什么？"→ `a[2] = 100;`（第 3 个格子编号是 2！）
- "5 个格子的储物柜，最后一个编号是几？"→ 4

### 4.5 用 for 循环操作数组（讲解②）

> 同学们，如果有 100 个格子，总不能写 100 行 `a[0]=... a[1]=...` 吧？
> 还记得我们的好朋友 **for 循环** 吗？它最擅长"重复做一件事"。

**输入数组：**

```cpp
int n;
cin >> n;                  // 先读入：一共几个数？
for (int i = 0; i < n; i++) {
    cin >> a[i];           // 把键盘输入的数放进 i 号柜
}
```

**输出数组：**

```cpp
for (int i = 0; i < n; i++) {
    cout << a[i] << " ";  // 依次打开每个柜子，展示里面的数
}
```

> 🔑 **黄金口诀**：
> ```
> "i 从 0 起步，i < n 停下，每次 i 加 1。"
> ```
> 这三句话，以后写数组循环闭着眼睛都要会背！

### 4.6 常见错误演示（讲解③）

用"找茬"游戏的方式，让学生当小侦探找 Bug：

**Bug 1：下标从 1 开始**
```cpp
for (int i = 1; i <= n; i++)  // ❌ 第 0 号柜被跳过了！
```

**Bug 2：数组越界**
```cpp
int a[5];
a[5] = 100;  // ❌ 只有 0~4 号柜，5 号柜不存在！
```

**Bug 3：数组大小不够**
```cpp
int a[5];
// 然后读入 10 个数 → 柜子不够用，程序会出错！
```

> 💡 小技巧：比赛中，我们习惯把数组开大一点，比如题目说"最多 100 个数"，我们就写 `int a[105];`，多留几个格子保平安。

### 4.7 课堂总结口诀

带着学生一起念：

> **数组就是储物柜，方括号里填大小；**
> **下标从零不从一，最大编号减个一；**
> **输入输出用循环，i 从 0 到 n 减 1。**

## 五、课堂例题

### 例题 1：初识数组——存 5 个同学的成绩并输出（教师演示）

**题目描述：**
输入 5 个同学的数学成绩，然后按顺序输出。

**讲解思路：**
1. 先声明一个大小为 5 的数组。
2. 用 for 循环逐个读入。
3. 再用 for 循环逐个输出。

```cpp
#include <iostream>
using namespace std;

int main() {
    int a[5]; // 造 5 个格子的储物柜

    // 输入：把成绩一个个放进柜子
    for (int i = 0; i < 5; i++) {
        cin >> a[i];
    }

    // 输出：把柜子里的成绩一个个拿出来看
    for (int i = 0; i < 5; i++) {
        cout << a[i] << " ";
    }
    cout << endl;

    return 0;
}
```

**运行示例：**
```
输入：88 95 73 60 100
输出：88 95 73 60 100
```

**教师讲解要点：**
- 逐行对照储物柜比喻讲解
- 强调 `i < 5` 而不是 `i <= 5`
- 让学生预测：如果改成 `i <= 5` 会怎样？

---

### 例题 2：先读个数，再读数据——更通用的写法（学生跟练）

**题目描述：**
第一行输入一个整数 n，表示有几个数；第二行输入 n 个整数。请按顺序输出这 n 个数。

**讲解思路：**
1. 把数组开大一点（比如 105），因为不知道 n 具体是多少。
2. 先读 n，再用 for 循环读 n 个数。
3. 用 for 循环输出。

```cpp
#include <iostream>
using namespace std;

int a[105]; // 开在 main 外面，开大一点，养成好习惯

int main() {
    int n;
    cin >> n;

    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    for (int i = 0; i < n; i++) {
        cout << a[i] << " ";
    }
    cout << endl;

    return 0;
}
```

**运行示例：**
```
输入：
4
10 20 30 40
输出：
10 20 30 40
```

**教师讲解要点：**
- 为什么数组开在 `main` 外面？→ 简单回答：放在外面可以开更大的数组，比赛常用写法。
- 为什么写 `a[105]` 而不是 `a[100]`？→ 多留几个"安全格子"。

---

### 例题 3：倒序输出数组（进阶体验）

**题目描述：**
输入 n 个整数，将它们**倒过来**输出。

**讲解思路：**
1. 先把所有数读进数组。
2. 输出时，for 循环**从后往前**走：`i` 从 `n-1` 到 `0`。

```cpp
#include <iostream>
using namespace std;

int a[105];

int main() {
    int n;
    cin >> n;

    // 正着读入
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    // 倒着输出
    for (int i = n - 1; i >= 0; i--) {
        cout << a[i] << " ";
    }
    cout << endl;

    return 0;
}
```

**运行示例：**
```
输入：
5
1 2 3 4 5
输出：
5 4 3 2 1
```

**教师讲解要点：**
- 画一排格子，手指从右往左指，帮助学生理解 `i = n-1; i >= 0; i--`。
- 提问："为什么 `i` 从 `n-1` 开始而不是 `n`？"→ 因为最后一个格子编号是 `n-1`。
- 🎉 **成就感设计**：这道题完成后表扬学生："你们已经会让电脑把一排数字倒过来了，这可是很多大人都觉得难的事情！"

### 练习 3：间隔输出——输出奇数位置的元素（⭐⭐⭐）

**题目：**
输入一个整数 n（n ≤ 100），接下来输入 n 个整数，存入数组。请输出数组中**下标为偶数**（0, 2, 4, ...）的元素。

**样例输入：**
```
6
10 20 30 40 50 60
```
**样例输出：**
```
10 30 50
```

**提示：** for 循环的第三部分可以写 `i += 2`。

**参考答案：**
```cpp
#include <iostream>
using namespace std;

int a[105];

int main() {
    int n;
    cin >> n;

    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    for (int i = 0; i < n; i += 2) {
        cout << a[i] << " ";
    }
    cout << endl;

    return 0;
}
```

## 七、常见错误与应对

| # | 常见错误 | 错误代码示例 | 正确写法 | 应对方法 |
|---|---------|------------|---------|---------|
| 1 | **下标从 1 开始** | `for(int i=1; i<=n; i++)` | `for(int i=0; i<n; i++)` | 反复强调口诀："i 从 0 起步，i < n 停下"；板书画格子编号 |
| 2 | **数组越界访问** | `int a[5]; a[5]=10;` | `a[4]=10;` | 画 5 个格子，标上 0~4，让学生数：没有 5 号！ |
| 3 | **数组开太小** | `int a[10];` 却读入 100 个数 | `int a[105];`（开大一些） | 教"多开 5 个"的小技巧，养成习惯 |
| 4 | **忘记读入 n** | 直接写 `for(int i=0; i<n; ...)` 但 n 没有 cin | 在循环前加 `cin >> n;` | 提醒学生：电脑不会读心术，你得告诉它 n 是几 |
| 5 | **输入输出用错循环变量** | `cin >> a[0];`（循环里写死了 0） | `cin >> a[i];` | 演示错误效果：每次都往 0 号柜塞东西，前面的被覆盖了 |
| 6 | **倒序循环条件写错** | `for(int i=n; i>0; i--)` | `for(int i=n-1; i>=0; i--)` | 用手指从最后一个格子开始指，数给学生看 |
