## 二、教学重点与难点

### 重点
- 三种数据类型的声明与输入输出语法。
- 理解"不同的盒子装不同的东西"——类型决定了变量能存什么。

### 难点
- `double` 与 `int` 的区别：为什么 `int` 装不了小数？装了会怎样？
- `char` 只能存一个字符，且声明时用**单引号**，容易与 `string` / 双引号混淆。
- 输入输出时不同类型的行为差异（如 `int` 会截断小数部分）。

## 四、讲稿提纲

### 4.1 热身复习（5 min）

> 同学们好！上节课我们学了一个超级重要的东西——**变量**。谁还记得变量是什么？
>
> 对！变量就像一个**盒子**，可以往里面装东西。上节课我们的盒子里装的都是**整数**，比如年龄、个数。我们用 `int` 来声明这种盒子。
>
> 快速回忆一下：
> ```cpp
> int age;
> cin >> age;
> cout << age << endl;
> ```
> 这三行代码做了什么？——①造了一个整数盒子 ②让用户往盒子里放一个数 ③把盒子里的数显示出来。
>
> 但是同学们想一想：如果我要存你的**身高 1.35 米**，用 `int` 行不行？如果我要存你名字的**第一个字母 'L'**，用 `int` 行不行？
>
> 今天，我们就来认识两种新盒子！

### 4.2 新课讲解①——int 复习 + double 新学（15 min）

#### int 快速复习（3 min）

> `int` 就是**整数盒子**，只能装整数：1、2、100、-5……没有小数点。
>
> 就好比一个存钱罐，只能放**硬币**（1元、5角这种整个整个的），你不能把半个硬币塞进去。

#### double 引入（12 min）

> 现在问题来了：数学课上我们会遇到 3.14、1.5、0.75 这些带小数点的数。`int` 盒子装不下它们！
>
> 所以 C++ 给了我们另一种盒子，叫 **`double`**（读作"达布偶"）。它就是**小数盒子**，专门装带小数点的数。
>
> 你可以把它想象成一个**量杯**——量杯上有刻度线，可以量出 1.5 升、2.3 升这种精确的数值。

**演示代码 1：double 的声明与输出**

```cpp
#include <iostream>
using namespace std;

int main() {
    double height = 1.35;
    cout << "我的身高是：" << height << " 米" << endl;
    return 0;
}
```

> 看，和 `int` 几乎一样！只是把 `int` 换成了 `double`，后面的数字可以带小数点了。

**演示代码 2：double 的输入**

```cpp
#include <iostream>
using namespace std;

int main() {
    double price;
    cout << "请输入商品价格：";
    cin >> price;
    cout << "价格是：" << price << " 元" << endl;
    return 0;
}
```

> `cin` 和 `cout` 聪明得很，它看到你的盒子是 `double`，就自动知道要读小数了。

**关键实验：int 盒子能不能装小数？**

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 3.9;
    cout << a << endl;  // 输出什么？
    return 0;
}
```

> 猜猜看输出什么？——是 **3**，不是 4！
>
> 因为整数盒子**只能装整数**，小数点后面的部分会被直接"砍掉"（不是四舍五入哦！）。就好比存钱罐的口太小了，小数部分塞不进去，掉地上了。
>
> 🔑 **记住**：int 会截断小数，不会四舍五入！

**小提示：double 也能装整数**

> `double price = 10;` 完全可以！小数盒子当然能装整数，就像量杯也可以装刚好 1 升的水。它会自动变成 10.0。

### 4.3 动手练习①（10 min）

> 同学们打开电脑，写一个小程序：
> - 声明一个 `double` 变量叫 `weight`
> - 用 `cin` 输入你的体重（比如 32.5）
> - 用 `cout` 输出："我的体重是 XX 千克"

（老师巡视，帮助遇到问题的同学）

### 4.4 新课讲解②——char 字符类型（15 min）

> 现在我们认识了整数盒子（int）和小数盒子（double），但世界上不只有数字！
>
> 如果我想在程序里存一个**字母**、一个**符号**，比如你成绩的等级 'A'、'B'、'C'，怎么办？
>
> C++ 给了我们第三种盒子——**`char`**（读作"卡"，character 字符的缩写）。它是**字符盒子**，可以装**一个字符**。

> 注意：char 盒子很小，一次只能装**一个字符**！
>
> 就好比一个小小的钥匙扣收纳盒，只能放**一把钥匙**。你不能往里面塞一串钥匙。

**演示代码 3：char 的声明与输出**

```cpp
#include <iostream>
using namespace std;

int main() {
    char grade = 'A';
    cout << "你的等级是：" << grade << endl;
    return 0;
}
```

> ⚠️ 特别注意：字符要用**单引号**括起来！`'A'` 不是 `"A"`。
>
> 怎么记？**单**引号装**单**个字符——都是"单"！

**演示代码 4：char 的输入**

```cpp
#include <iostream>
using namespace std;

int main() {
    char ch;
    cout << "请输入一个字母：";
    cin >> ch;
    cout << "你输入的是：" << ch << endl;
    return 0;
}
```

**趣味实验：char 的秘密——字符其实是数字！**

```cpp
#include <iostream>
using namespace std;

int main() {
    char ch = 'A';
    cout << ch << endl;          // 输出 A
    cout << (int)ch << endl;     // 输出 65
    return 0;
}
```

> 哇！字母 'A' 在电脑里其实是数字 **65**！每个字符都有一个对应的编号，这叫 **ASCII 码**。
>
> 就像每个同学在班级里都有一个学号一样，每个字符在电脑里也有一个编号。
>
> - 'A' = 65, 'B' = 66, 'C' = 67 …
> - 'a' = 97, 'b' = 98, 'c' = 99 …（小写字母编号不一样哦！）
> - '0' = 48, '1' = 49 …（字符 '0' 不是数字 0 哦！）
>
> 现在只需要知道有这回事就行，以后会经常用到。

### 4.5 动手练习②（10 min）

> 同学们试一试：
> 1. 声明一个 `char` 变量，赋值为你名字拼音的第一个字母，输出它。
> 2. 再试试输出 `(int)` 你的字母，看看它的 ASCII 编号是多少！

### 4.6 休息（5 min）

> 站起来伸个懒腰！活动一下手指！

### 4.7 综合讲解——三种类型大对比（15 min）

> 我们来做一个总结表格：

| 类型 | 中文名 | 比喻 | 能装什么 | 例子 | 引号 |
|------|--------|------|----------|------|------|
| `int` | 整数类型 | 存钱罐 | 整数 | 1, -5, 100 | 不需要 |
| `double` | 小数类型 | 量杯 | 整数和小数 | 3.14, -0.5, 10.0 | 不需要 |
| `char` | 字符类型 | 钥匙扣盒 | 一个字符 | 'A', 'b', '3', '+' | **单引号** |

**场景选择小游戏（互动）**

> 老师说一个东西，你们告诉我应该用什么盒子装：
>
> 1. 你的年龄（10岁）→ ？ → **int**！
> 2. 一瓶可乐的价格（3.5元）→ ？ → **double**！
> 3. 你考试的等级（'A'）→ ？ → **char**！
> 4. 全班人数（45人）→ ？ → **int**！
> 5. 圆周率（3.14159）→ ？ → **double**！
> 6. 红绿灯的状态（'R' / 'G' / 'Y'）→ ？ → **char**！
> 7. 你的身高（1.42米）→ ？ → **double**！
> 8. 一个数学运算符号（'+'）→ ？ → **char**！

**综合演示代码：三种类型一起用**

```cpp
#include <iostream>
using namespace std;

int main() {
    // 声明三种盒子
    int age;
    double height;
    char grade;

    // 输入
    cout << "请输入你的年龄：";
    cin >> age;
    cout << "请输入你的身高（米）：";
    cin >> height;
    cout << "请输入你的等级（A/B/C）：";
    cin >> grade;

    // 输出
    cout << "======== 个人信息卡 ========" << endl;
    cout << "年龄：" << age << " 岁" << endl;
    cout << "身高：" << height << " 米" << endl;
    cout << "等级：" << grade << endl;
    cout << "============================" << endl;

    return 0;
}
```

> 看！三种盒子和平共处，各司其职。`cin` 和 `cout` 会根据变量类型自动判断怎么读、怎么输出。

## 五、课堂例题

### 例题 1：自我介绍卡（入门）

**题目描述**：输入一个人的年龄（整数）、身高（小数）、姓名首字母（字符），输出一张个人信息卡。

**输入样例**：
```
10 1.42 L
```

**输出样例**：
```
年龄：10岁
身高：1.42米
姓名首字母：L
```

**完整代码**：

```cpp
#include <iostream>
using namespace std;

int main() {
    int age;
    double height;
    char initial;

    cin >> age >> height >> initial;

    cout << "年龄：" << age << "岁" << endl;
    cout << "身高：" << height << "米" << endl;
    cout << "姓名首字母：" << initial << endl;

    return 0;
}
```

**讲解思路**：
1. 先分析题目需要哪些变量：年龄是整数→`int`，身高是小数→`double`，首字母是字符→`char`。
2. 三个变量可以一行 `cin` 读入，用空格隔开。
3. 输出时按格式拼接就行。这道题是三种类型的"全家福"，帮同学们练习最基本的声明、输入、输出。

---

### 例题 2：小商店计算器（进阶）

**题目描述**：输入一件商品的单价（小数）和购买数量（整数），输出总价。

**输入样例**：
```
3.5 4
```

**输出样例**：
```
总价：14
```

**完整代码**：

```cpp
#include <iostream>
using namespace std;

int main() {
    double price;
    int count;

    cin >> price >> count;

    double total = price * count;
    cout << "总价：" << total << endl;

    return 0;
}
```

**讲解思路**：
1. 单价有小数→`double`，数量是整数→`int`。
2. `double` 和 `int` 可以直接相乘，结果是 `double`（小数盒子能装下结果）。
3. 这里引出一个小知识：**不同类型可以混合运算**，C++ 会自动把 `int` 临时"升级"成 `double` 来计算，这样就不会丢精度。就好像量杯和存钱罐里的东西一起算，结果放到量杯里更安全。

---

### 例题 3：字符的前后邻居（拓展）

**题目描述**：输入一个小写字母，输出它的前一个字母和后一个字母。

**输入样例**：
```
d
```

**输出样例**：
```
d的前一个字母是c
d的后一个字母是e
```

**完整代码**：

```cpp
#include <iostream>
using namespace std;

int main() {
    char ch;
    cin >> ch;

    char prev = ch - 1;
    char next = ch + 1;

    cout << ch << "的前一个字母是" << prev << endl;
    cout << ch << "的后一个字母是" << next << endl;

    return 0;
}
```

**讲解思路**：
1. 因为字符在电脑里就是数字（ASCII码），所以字符可以做加减法！
2. `'d'` 的编号是 100，`'d' - 1` 就是 99，对应 `'c'`；`'d' + 1` 就是 101，对应 `'e'`。
3. 这就像找你在班级座位表里前面和后面的同学一样——你的学号减1就是前一个同学，加1就是后一个同学。
4. 这道题为后续使用 ASCII 码做字符处理打下基础。

## 七、常见错误与应对

### 错误 1：double 变量写成了 int

```cpp
int price = 3.5;    // ❌ 错误！结果是 3，小数部分被截断了
double price = 3.5;  // ✅ 正确！
```

**应对**：提醒学生——"装小数要用量杯（double），用存钱罐（int）会把小数部分弄丢！"

---

### 错误 2：char 用了双引号

```cpp
char grade = "A";   // ❌ 错误！双引号是字符串，不是字符
char grade = 'A';   // ✅ 正确！单引号才是字符
```

**应对**：口诀——"**单**引号装**单**个字符，都是'单'！双引号是装一串字的，以后再学。"

---

### 错误 3：char 里放了多个字符

```cpp
char name = 'Tom';  // ❌ 错误！char 只能放一个字符
char initial = 'T'; // ✅ 正确！
```

**应对**：提醒学生——"钥匙扣盒只能放一把钥匙！要放一串字以后学 string。"

---

### 错误 4：变量类型名拼写错误

```cpp
dubble x = 3.14;    // ❌ 拼写错误！是 double 不是 dubble
duoble x = 3.14;    // ❌ 拼写错误！
```

**应对**：在黑板上大写板书 `d-o-u-b-l-e`，带全班拼读两遍。可以谐音记忆："达-布-偶"。

---

### 错误 5：忘记 `endl` 或输出格式混乱

```cpp
cout << age << height;  // 输出 101.42，数字黏在一起分不清
```

**应对**：提醒加空格或换行，养成好习惯：`cout << age << " " << height << endl;`

---

### 错误 6：cin 输入类型不匹配

比如程序期望输入 `int`，用户却输了一个字母，程序会出错。

**应对**：告诉学生——"现在先保证自己输入正确的数据类型。就像存钱罐只能投硬币，你往里面塞纸条它会卡住的。"
