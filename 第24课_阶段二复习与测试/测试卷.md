## 二、改错题（每题 10 分，共 20 分）

### 第 6 题 答案

**错误 1**：第 4 行 `void maxVal(int a, int b)` 的返回类型应为 `int`，不能是 `void`

> 说明：函数体内有 `return a` 和 `return b`，要返回整数值，返回类型必须是 `int`。应改为 `int maxVal(int a, int b)`

**错误 2**：由于返回类型写成了 `void`，`void` 类型的函数不能出现在 `cout <<` 中输出。修正错误 1 后此问题自动解决。但本质上这是两个互相关联的错误：① 返回类型错误 ② 无法输出 `void` 函数的返回值。

> **评分说明**：本题的两处错误都指向同一个根本原因——返回类型写错。如果学生只找到一处并修正了 `void` → `int`，可给 7-8 分。如果学生能说清 `void` 函数不能 return 值，且不能被 cout 输出，给满分。

| 得分点 | 分值 |
|--------|------|
| 找到 `void` 应改为 `int` 的问题 | 4 分 |
| 正确解释 `void` 函数不能用 `return` 返回值 | 3 分 |
| 正确解释 `void` 函数的调用不能用在 `cout` 中 | 3 分 |

---

### 第 7 题 答案

**错误 1**：第 8 行 `for (int j = 0; j < n - 1; j++)` 应改为 `for (int j = 0; j < n - 1 - i; j++)`

> 说明：冒泡排序每完成一轮，最后面就多一个已经排好的元素，内层循环上界应该逐轮减少。不改的话程序仍能运行但效率低（这里也可以不算错，取决于考试要求）。

**错误 2**：第 9 行 `if (a[j] < a[j + 1])` 应改为 `if (a[j] > a[j + 1])`

> 说明：题目要求从小到大排序，应当在前面的数**大于**后面的数时交换。用 `<` 会排成从大到小。

| 得分点 | 分值 |
|--------|------|
| 找到错误 1（内层循环上界优化）的位置 | 2 分 |
| 正确说明错误 1 的原因并修改 | 3 分 |
| 找到错误 2（比较方向反了）的位置 | 2 分 |
| 正确说明错误 2 的原因并修改 | 3 分 |

> 💡 **给老师的提示**：错误 1 在某些评判标准下不算"错误"（只是不够优化），如果学生找到了错误 2，且将错误 1 指出为"内层循环可以优化为 `n-1-i`"，也应给满分。如果学生只找到错误 2，给 5-6 分。

### 第 10 题 参考答案（22 分）

```cpp
#include <iostream>
#include <cstring>
using namespace std;

int main() {
    string s1, s2;
    cin >> s1 >> s2;
    
    int a[210] = {0}, b[210] = {0}, c[210] = {0};
    int len1 = s1.length();
    int len2 = s2.length();
    
    // 倒序存入数组（个位在下标 0）
    for (int i = 0; i < len1; i++) {
        a[i] = s1[len1 - 1 - i] - '0';
    }
    for (int i = 0; i < len2; i++) {
        b[i] = s2[len2 - 1 - i] - '0';
    }
    
    // 逐位相加 + 进位
    int len = max(len1, len2);
    for (int i = 0; i < len; i++) {
        c[i] += a[i] + b[i];
        c[i + 1] += c[i] / 10;   // 进位
        c[i] %= 10;               // 保留个位
    }
    if (c[len] > 0) len++;        // 最高位有进位
    
    // 倒序输出
    for (int i = len - 1; i >= 0; i--) {
        cout << c[i];
    }
    cout << endl;
    
    return 0;
}
```

| 逐位相加逻辑正确 | 4 分 |
| 进位处理正确 | 4 分 |
