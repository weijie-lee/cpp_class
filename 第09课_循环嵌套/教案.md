## 二、教学重点与难点

### 重点

- 循环嵌套的概念：**外层循环控制"行"，内层循环控制"列"**。
- 内层循环每次都要"从头再来"——外层每走一步，内层完整跑一轮。
- 九九乘法表的实现：内层终点 `j <= i`，体现内外层变量的依赖关系。

### 难点

- 理解"循环里面套循环"的执行流程，不要把两层循环搞混。
- 内层循环的终止条件**随外层变量变化**（如三角形：第 i 行打印 i 个星号）。
- `endl` 的位置：换行应写在**内层循环结束之后、外层循环体内**。

## 四、讲稿提纲

### 4.1 热身引入（8 min）

#### 比喻一：时钟的时针和分针

> 同学们，你们看过钟表吧？钟上面有两根针——时针和分针。
>
> 时针走得**慢**，指到"1点"之后就不动了，等分针呢？分针"滴答滴答"从 0 跑到 59，跑完一整圈！然后时针才慢悠悠地挪到"2点"，分针又从 0 开始重新跑一整圈……
>
> - 时针就是**外层循环**——走得慢，每走一步要等里面跑完。
> - 分针就是**内层循环**——跑得快，每次从头跑到尾。
>
> 时针从 1 走到 12，每个小时里分针都跑了 60 下。一天下来分针总共跑了 12 × 60 = 720 下！

**互动提问**：时针走到 3 点的时候，分针跑了几圈？（3 圈！）总共走了多少下？（3 × 60 = 180 下！）

#### 比喻二：体育课跑操

> 再想一个场景：运动会入场式，4 个班级排队进场。
>
> 老师喊：**"第 1 个班，进场！"** 然后这个班的 5 个同学**一个一个**走过主席台——第 1 个同学、第 2 个……第 5 个，走完了。
>
> 老师又喊：**"第 2 个班，进场！"** 又是 5 个同学一个一个走过去……
>
> 这里，**班级是外层循环**（走 4 步，每步对应一个班），**班里的同学是内层循环**（每个班里从第 1 个走到第 5 个）。
>
> 每换一个班（外层循环走一步），同学编号都要**从第 1 个重新开始**！

**互动提问**：如果有 3 个班，每个班 6 个人，一共有多少个同学走过主席台？（3 × 6 = 18 个！）

> 今天我们就来学——**循环嵌套**：在一个循环里面，再放一个循环！

---

### 4.2 循环嵌套语法与执行流程（17 min）

#### 板书核心结构

```
for (外层初始化; 外层条件; 外层更新) {
    // 外层循环体——"每个班"
    for (内层初始化; 内层条件; 内层更新) {
        // 内层循环体——"每个同学"
    }
    // 内层跑完一轮后，还可以做点事（比如换行）
}
```

> 翻译成大白话就是：
>
> ```
> for (从第1个班开始; 班级还没排完; 下一个班) {
>     for (从第1个同学开始; 同学还没走完; 下一个同学) {
>         这个同学走过主席台;
>     }
>     这个班走完了，空一行;
> }
> ```

#### 手动模拟：2 个班，每班 3 个同学

翻译成 C++ 代码：

```cpp
for (int i = 1; i <= 2; i++) {        // i: 班级编号
    for (int j = 1; j <= 3; j++) {    // j: 同学编号
        cout << "第" << i << "班第" << j << "号 ";
    }
    cout << endl;  // 一个班走完了，换行
}
```

**请一位同学上台当"计算机"，老师喊口令，走模拟表：**

| 外层 i | 内层 j | 输出 |
|--------|--------|------|
| 1 | 1 | 第1班第1号 |
| 1 | 2 | 第1班第2号 |
| 1 | 3 | 第1班第3号 |
| 1 | （j 变 4，不满足 j<=3，内层结束） | **换行** |
| 2 | 1 | 第2班第1号 |
| 2 | 2 | 第2班第2号 |
| 2 | 3 | 第2班第3号 |
| 2 | （j 变 4，不满足 j<=3，内层结束） | **换行** |

输出效果：

```
第1班第1号 第1班第2号 第1班第3号
第2班第1号 第2班第2号 第2班第3号
```

> 🎯 **关键总结**（板书大字）：
>
> - 外层循环走**一步**，内层循环跑**一整轮**。
> - 内层跑完一轮后，回到外层，外层再走一步，内层又从头开始。
> - 总执行次数 = 外层次数 × 内层次数（2 × 3 = 6 次）。

#### `endl` 放在哪里？

> 特别注意：`endl`（换行）要放在**内层循环的外面、外层循环的里面**。
>
> 想象你在写作文：每个字（内层循环）写在同一行，写完一行后才按"回车键"换到下一行（`endl`），然后开始写新的一行。

## 五、课堂例题

### 例题 1：打印矩形星号阵（最基础的嵌套，建立信心）

**题目**：输入两个正整数 n 和 m，打印一个 n 行 m 列的星号矩形。

```cpp
#include <iostream>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    for (int i = 1; i <= n; i++) {       // 外层：控制行数
        for (int j = 1; j <= m; j++) {   // 内层：控制每行的星号个数
            cout << "* ";
        }
        cout << endl;  // 一行打完了，换行
    }
    return 0;
}
```

**样例输入**：`3 5`

**样例输出**：

```
* * * * *
* * * * *
* * * * *
```

**讲解思路**：

1. 先问学生：如果只打印 1 行 5 个星号，怎么写？→ 一个 for 就够了。
2. 那如果要打印 3 行呢？每行都是"打印 5 个星号然后换行"，这个动作要**重复 3 次**——外面再套一个循环！
3. 强调 `i` 和 `j` 是两个不同的循环变量，`i` 管行，`j` 管列。就像跑操时，`i` 代表第几个班，`j` 代表班里的第几个同学。
4. **现场演示**：把 `endl` 移到内层循环里面，运行看看会变成什么样——所有星号都变成一列了！帮学生理解 `endl` 位置的重要性。

---

### 例题 2：打印直角三角形（内层终点依赖外层变量）

**题目**：输入正整数 n，打印一个 n 行的直角三角形。

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {       // 第 i 行
        for (int j = 1; j <= i; j++) {   // 第 i 行打印 i 个星号
            cout << "* ";
        }
        cout << endl;
    }
    return 0;
}
```

**样例输入**：`5`

**样例输出**：

```
*
* *
* * *
* * * *
* * * * *
```

**讲解思路**：

1. **和例题 1 对比**：唯一的变化是内层循环的终点从 `m`（固定值）变成了 `i`（跟着外层变）！

2. **生活比喻：叠积木**
   > 想象你在叠积木塔：第 1 层放 1 块，第 2 层放 2 块，第 3 层放 3 块……第 i 层放 i 块。
   > 每一层放多少块，取决于你现在是第几层——**内层循环的次数由外层的 `i` 决定**。

3. 手动模拟前 3 行：

   | 外层 i | 内层 j 的范围 | 打印星号个数 | 输出 |
   |--------|-------------|------------|------|
   | 1 | j: 1→1 | 1 个 | `*` |
   | 2 | j: 1→2 | 2 个 | `* *` |
   | 3 | j: 1→3 | 3 个 | `* * *` |

4. 关键提问：第 4 行打印几个星号？（4 个！）最后一行（第 n 行）打印几个？（n 个！）

---

### 例题 3：九九乘法表（经典中的经典）

**题目**：打印九九乘法表。

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 9; i++) {           // i: 行号，也是第一个乘数
        for (int j = 1; j <= i; j++) {       // j: 列号，也是第二个乘数
            cout << j << "*" << i << "=" << i * j << "\t";
        }
        cout << endl;
    }
    return 0;
}
```

**样例输出**（部分）：

```
1*1=1
1*2=2	2*2=4
1*3=3	2*3=6	3*3=9
1*4=4	2*4=8	3*4=12	4*4=16
...
1*9=9	2*9=18	3*9=27	4*9=36	5*9=45	6*9=54	7*9=63	8*9=72	9*9=81
```

**讲解思路**：

1. **先建立直觉**：在黑板上写出九九乘法表的前 3 行，问学生——
   > 看，第 1 行只有 1 个算式，第 2 行有 2 个，第 3 行有 3 个……第 i 行有 i 个！是不是和刚才的三角形一样？

2. **类比三角形**：三角形每行打印 `i` 个星号，乘法表每行打印 `i` 个算式。框架完全一样，只是循环体里"打印什么"不同。

3. **内容拆解**：每个算式的格式是 `j * i = i*j`。
   > `j` 是小的那个数（列号），`i` 是大的那个数（行号），结果是 `i * j`。

4. **新知识：`\t`（制表符）**：
   > `\t` 就像按一下键盘上的 Tab 键，能让输出**自动对齐**。如果不用 `\t`，乘法表会挤在一起，很难看。

5. **现场互动**：先让学生猜第 5 行有几个算式（5 个），最后一个算式是什么（`5*5=25`）。

---

### 例题 4：打印倒直角三角形（反向思考）

**题目**：输入正整数 n，打印一个倒直角三角形。

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    for (int i = n; i >= 1; i--) {       // 从第 n 行开始，倒着来
        for (int j = 1; j <= i; j++) {   // 第 i 行打印 i 个星号
            cout << "* ";
        }
        cout << endl;
    }
    return 0;
}
```

**样例输入**：`5`

**样例输出**：

```
* * * * *
* * * *
* * *
* *
*
```

**讲解思路**：

1. **和例题 2 对比**：正三角形是 `i` 从 1 增长到 n，倒三角形是 `i` 从 n 减小到 1。
2. 第 7 课学过"倒数"（`i--`），现在用到嵌套里了——**旧知识在新场景中复用**。
3. 还有另一种思路：外层 `i` 仍然从 1 到 n，但内层终点写成 `n - i + 1`。两种都可以，现场对比一下。
4. **成就感设计**：让学生自己先试，能写出来的同学举手，老师记名表扬。

```cpp
            cout << num << " ";

    }
    return 0;
}
```

## 七、常见错误与应对

### 错误 1：内外层循环用了同一个变量名

```cpp
// ❌ 错误：两层循环都用 i，内层会把外层的 i 搞乱！
for (int i = 1; i <= 3; i++) {
    for (int i = 1; i <= 3; i++) {   // 又定义了一个 i
        cout << "* ";
    }
    cout << endl;
}
```

**现象**：可能编译通过但逻辑混乱，或得到意想不到的结果。

**应对**：

> 内外层循环就像两个人做不同的工作，一个管行、一个管列，他们的名字不能一样！
> **约定：外层用 `i`，内层用 `j`**，如果还有第三层就用 `k`——就像数学课上用 x、y、z 一样。

---

### 错误 2：`endl` 写在了内层循环里面

```cpp
// ❌ 错误：每打一个星号就换行了！
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        cout << "* ";
        cout << endl;    // ← 写错位置了！
    }
}
```

**现象**：每个星号都单独占一行，打出来变成了一长列。

**应对**：

> 想象你在写字：每写一个字（内层循环）不要按回车！要等**一整行字都写完**（内层循环结束），才按一次回车。
> `endl` 必须放在**内层 for 的外面、外层 for 的里面**。

---

### 错误 3：内层循环的终点写错，忘了和外层变量关联

```cpp
// 想打印三角形，但内层终点写成了固定值 n
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= n; j++) {   // ← 应该是 j <= i，不是 j <= n！
        cout << "* ";
    }
    cout << endl;
}
```

**现象**：打出来是矩形，不是三角形。

**应对**：

> 你想要三角形：第 1 行 1 个，第 2 行 2 个……第 i 行 i 个。
> 那内层循环的终点就应该是 `i`，而不是 `n`。
> **记住：三角形的秘密在于——内层的终点跟着外层走。**

---

### 错误 4：九九乘法表的列号和行号搞反

```cpp
// ❌ 输出 i*j 但 i 在前 j 在后，结果没问题但格式不标准
cout << i << "*" << j << "=" << i * j << "\t";
```

**现象**：输出变成了 `1*1=1  2*1=2  3*1=3 ...`，看着不对劲。

**应对**：

> 乘法表的习惯写法是**小数在前、大数在后**。
> 每一行里 `j` 从 1 跑到 `i`，所以 `j` 是小的，`i` 是大的——先写 `j`，再写 `i`。

---

### 错误 5：忘了外层循环，只写了一层

```cpp
// ❌ 只打印了一行星号
for (int j = 1; j <= 5; j++) {
    cout << "* ";
}
cout << endl;
// 忘记外面还需要一层循环来控制行数了！
```

**现象**：只输出了一行。

**应对**：

> 看到题目说"打印 n **行**"，脑子里立刻响起一个声音——**"行"用外层循环，"列"用内层循环**。先把外层的框架搭好，再在里面填内层循环。

## 九、下节课预告

> 下节课我们学 **一维数组** —— 让计算机一次记住一排数字！
>
> 想象你的书桌上有一排抽屉，编号 0、1、2、3、4……每个抽屉里可以放一个数字。你可以说"打开第 3 号抽屉，看看里面是什么"，也可以说"把 100 放进第 5 号抽屉"。
>
> 有了数组，我们就能处理**一大批数据**了——比如记录全班 40 个同学的考试成绩，然后用循环一个一个读出来、加起来、算平均分。
>
> 这可是竞赛里最最基础的"数据容器"，非常重要，期待下节课见！😊
