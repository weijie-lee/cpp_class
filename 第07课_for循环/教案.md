## 二、教学重点与难点

### 重点

- 循环的核心思想：**让计算机帮我们重复做同一件事**。
- for 语句的语法结构及三个部分各自的职责。
- 循环变量 `i` 从哪里开始、什么时候停、每一步怎么变。

### 难点

- 理解"条件为 true 才继续跑"——什么时候停下来？差一错误（off-by-one）。
- 循环体内的累加思想（`sum = sum + i`），变量在循环中反复被修改。

## 四、讲稿提纲

### 4.1 热身引入（5 min）

> 同学们，你们体育课跑过操场吗？老师说"跑 5 圈"，你们是不是要——
> 第 1 圈、第 2 圈、第 3 圈、第 4 圈、第 5 圈……跑完 5 圈才停下来？
>
> 想象你手上有一个**计数器**（就像那种菜市场阿姨用的计数器，按一下数字加 1）。
> - 起跑前把计数器**拨到 1**（这是"初始化"）
> - 每跑完一圈，看一眼计数器：**数字是不是还 ≤ 5？** 如果是，继续跑；如果超了，停！（这是"条件"）
> - 每跑完一圈，按一下计数器，**数字 + 1**（这是"更新"）
>
> 计算机也是一样的！它跑得比你快一万倍，但它**很笨**，不告诉它"重复"，它不会重复。
> 今天我们就来学一个魔法口令—— **for 循环**，让计算机帮我们"跑圈"！

**互动提问**：生活中还有哪些"重复做同一件事"的例子？（抄写生词 10 遍、每天刷牙、打印 100 份试卷……）

---

### 4.2 for 循环语法（15 min）

#### 板书核心结构

```
for (初始化; 条件; 更新) {
    循环体：要重复做的事
}
```

> 咱们把它对应到跑圈：
>
> ```
> for (计数器拨到1; 计数器还≤5吗; 按一下计数器加1) {
>     跑一圈；
> }
> ```
>
> 翻译成 C++ 就是：
>
> ```cpp
> for (int i = 1; i <= 5; i++) {
>     cout << "我跑了第 " << i << " 圈" << endl;
> }
> ```

#### 执行流程（重点！慢慢走一遍）

用"**四步舞**"来记：

| 步骤 | 动作 | 对应代码 |
|------|------|----------|
| 第 1 步 | 只做一次：设好起点 | `int i = 1` |
| 第 2 步 | 检查条件：能不能继续？ | `i <= 5` → 能！ |
| 第 3 步 | 做事情（循环体） | `cout << ...` |
| 第 4 步 | 更新计数器 | `i++`（i 变成 2） |
| 然后回到第 2 步…… | 检查 → 做事 → 更新 → 检查 → …… | 直到条件不满足 |

> **注意**：第 1 步只走一次！后面一直是 2→3→4→2→3→4→……直到第 2 步说"不行了，停！"

**手动模拟**（请一位同学上台当"计算机"，老师喊口令）：

| 轮次 | i 的值 | i <= 5？ | 输出 | i++ 后 |
|------|--------|----------|------|--------|
| 第 1 轮 | 1 | ✅ 是 | "我跑了第 1 圈" | i 变 2 |
| 第 2 轮 | 2 | ✅ 是 | "我跑了第 2 圈" | i 变 3 |
| 第 3 轮 | 3 | ✅ 是 | "我跑了第 3 圈" | i 变 4 |
| 第 4 轮 | 4 | ✅ 是 | "我跑了第 4 圈" | i 变 5 |
| 第 5 轮 | 5 | ✅ 是 | "我跑了第 5 圈" | i 变 6 |
| 第 6 轮 | 6 | ❌ 不是 | 不输出，循环结束 | — |

> 🎯 关键一问：循环跑了几圈？**5 圈。** `i` 最后变成了几？**6**，但第 6 轮一检查就停了，没有执行循环体。

### 4.3 `i` 是什么？（补充讲解 2 min）

> `i` 就是个**变量名**，代表 "index"（编号）。你也可以叫它 `a`、`x`、`round`，但大家约定俗成用 `i`，就像跑步时的记圈器一样。
>
> `i++` 是 `i = i + 1` 的简写，还记得上节课学的运算符吗？

## 五、课堂例题

### 例题 1：输出 1 到 5（最简循环，建立信心）

**题目**：输出 1、2、3、4、5，每个数占一行。

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << i << endl;
    }
    return 0;
}
```

**输出**：

```
1
2
3
4
5
```

**讲解思路**：

1. 先问学生：如果不用循环，怎么写？→ 写 5 行 `cout`。
2. 如果要输出 1 到 100 呢？写 100 行吗？太累了！→ 引出循环的价值。
3. 逐行对照"四步舞"，带学生走一遍执行过程。
4. 现场改数字：把 5 改成 10，运行给学生看——**改一个数，多跑 5 圈，循环的威力！**

---

### 例题 2：输出 1 到 N（从固定到灵活）

**题目**：输入一个整数 N，输出从 1 到 N 的所有整数，每个数占一行。

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cout << i << endl;
    }
    return 0;
}
```

**样例输入**：`7`

**样例输出**：

```
1
2
3
4
5
6
7
```

**讲解思路**：

1. 和例题 1 对比：**唯一的变化就是把 `5` 换成了变量 `n`**，循环的"终点"可以由用户决定了！
2. 强调 `cin >> n` 要写在 for 循环前面——先知道跑几圈，才能开跑。
3. 互动：如果输入 0，会输出什么？→ 一开始 `1 <= 0` 就是 false，**一圈都不跑**！

---

### 例题 3：求 1 到 N 的和（引入累加思想）

**题目**：输入正整数 N，计算 1 + 2 + 3 + … + N 的结果。

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int sum = 0;  // 准备一个"存钱罐"，一开始是空的
    for (int i = 1; i <= n; i++) {
        sum = sum + i;  // 每跑一圈，往存钱罐里放 i 块钱
    }
    cout << sum << endl;
    return 0;
}
```

**样例输入**：`5`

**样例输出**：`15`

**讲解思路**：

1. **生活比喻：存钱罐**。
   > 想象你有一个空的存钱罐（`sum = 0`）。
   > 第 1 天放 1 块钱，第 2 天放 2 块钱……第 5 天放 5 块钱。
   > 最后存钱罐里有多少钱？1+2+3+4+5 = **15 块**！

2. 手动模拟表格（板书）：

   | 轮次 | i | sum（进入时） | sum = sum + i（放钱后） |
   |------|---|--------------|----------------------|
   | 第 1 轮 | 1 | 0 | 0 + 1 = **1** |
   | 第 2 轮 | 2 | 1 | 1 + 2 = **3** |
   | 第 3 轮 | 3 | 3 | 3 + 3 = **6** |
   | 第 4 轮 | 4 | 6 | 6 + 4 = **10** |
   | 第 5 轮 | 5 | 10 | 10 + 5 = **15** |

3. 重点强调：**`sum` 一定要在循环外面初始化为 0**，不然存钱罐里一开始就有"脏数据"。

---

### 例题 4：输出 N 到 1 的倒数（循环变量可以递减）

**题目**：输入正整数 N，从 N 倒数到 1，每个数占一行。

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    for (int i = n; i >= 1; i--) {
        cout << i << endl;
    }
    return 0;
}
```

**样例输入**：`5`

**样例输出**：

```
5
4
3
2
1
```

**讲解思路**：

1. 和例题 2 对比：**起点从 1 变成 n，条件从 `<=` 变成 `>=`，更新从 `i++` 变成 `i--`**。
2. 比喻：火箭倒计时！5、4、3、2、1，发射！🚀
3. 关键问题：如果写成 `i <= 1`，会怎样？→ 一圈都不跑（除非 n 就是 1）！引导学生注意**条件和更新方向要匹配**。

## 六、课堂练习题

### 练习 1（基础·上手）

**题目**：输入一个正整数 N，输出从 1 到 N 的所有**偶数**，每个数占一行。

**提示**：想想 `i` 的起点和每次加多少。

**参考答案**：

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    for (int i = 2; i <= n; i = i + 2) {
        cout << i << endl;
    }
    return 0;
}
```

> 也可以用 `i++` 配合 `if (i % 2 == 0)` 来写，两种方法都对，鼓励学生讨论哪种更好。

---

### 练习 2（巩固·累加变形）

**题目**：输入正整数 N，计算 1 + 3 + 5 + … 的所有不超过 N 的奇数之和。

**样例输入**：`10`
**样例输出**：`25`（即 1+3+5+7+9 = 25）

**参考答案**：

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int sum = 0;
    for (int i = 1; i <= n; i = i + 2) {
        sum = sum + i;
    }
    cout << sum << endl;
    return 0;
}
```

---

### 练习 3（进阶·综合应用）

**题目**：输入正整数 N，输出 N 行如下图案（以 N=5 为例）：

```
*
**
***
****
*****
```

**提示**：第 `i` 行要输出 `i` 个星号——这需要**一个循环里面再套一个循环**吗？先试试只用一个循环行不行。

**参考答案（方法一：利用 string 构造）**：

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        // 用 string 的构造函数：重复 i 次字符 '*'
        cout << string(i, '*') << endl;
    }
    return 0;
}
```

**参考答案（方法二：循环嵌套，简单提一下即可）**：

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= i; j++) {
            cout << "*";
        }
        cout << endl;
    }
    return 0;
}
```

> 方法二用到了"循环嵌套"，这是第 9 课的内容。如果有学生问到，简单解释"for 里面再套一个 for"即可，不展开。

---

### 练习 4（挑战·选做）

**题目**：输入正整数 N，求 N 的阶乘（N! = 1 × 2 × 3 × … × N）。

**样例输入**：`5`
**样例输出**：`120`

**参考答案**：

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int result = 1;  // 注意：乘法的"存钱罐"从 1 开始，不是 0！
    for (int i = 1; i <= n; i++) {
        result = result * i;
    }
    cout << result << endl;
    return 0;
}
```

> 🎯 和累加对比：累加从 0 开始（加法的"零元素"），累乘从 1 开始（乘法的"幺元素"），引导学生思考为什么。

## 七、常见错误与应对

### 错误 1：分号写错位置

```cpp
// ❌ 错误写法：for 后面多了一个分号！
for (int i = 1; i <= 5; i++);   // ← 这个分号让循环体变成了"空"的！
{
    cout << i << endl;
}
```

**现象**：只输出一个数（或者编译报错找不到 `i`）。

**应对**：告诉学生——**for 那一行的末尾绝对不要加分号**，分号就像一堵墙，把循环体挡在了外面。

---

### 错误 2：循环条件和更新方向矛盾

```cpp
// ❌ 想从 5 数到 1，但 i++ 让 i 越来越大，永远停不下来！
for (int i = 5; i >= 1; i++) {
    cout << i << endl;
}
```

**现象**：程序一直跑，不停地输出数字（死循环）。

**应对**：口诀——**"往上数用 ++，往下数用 --"**。条件和更新方向要一致。

---

### 错误 3：差一错误（Off-by-one）

```cpp
// 想输出 1 到 5，但写成了 i < 5
for (int i = 1; i < 5; i++) {
    cout << i << endl;
}
// 输出：1 2 3 4   ← 少了个 5！
```

**应对**：

> 你想跑 5 圈，条件要"**跑到第 5 圈还能跑**"。
> - `i <= 5`：第 5 圈 ✅ 能跑，第 6 圈 ❌ 停。**跑了 5 圈。**
> - `i < 5`：第 5 圈就停了，只跑了 4 圈。**少了一圈！**
>
> 建议刚学时统一用 `i <= n` 这种写法，不容易搞混。

---

### 错误 4：忘记初始化累加变量

```cpp
int sum;  // ❌ 没有 = 0，sum 里面可能是任何奇怪的数！
for (int i = 1; i <= 5; i++) {
    sum = sum + i;
}
cout << sum << endl;  // 输出结果是错的
```

**应对**：

> 存钱罐一定要先**倒空**（`= 0`），不然里面不知道原来装了什么脏东西。

---

### 错误 5：大括号忘写或配对错误

```cpp
// ❌ 忘记大括号，只有第一句在循环里
for (int i = 1; i <= 5; i++)
    cout << i;
    cout << endl;  // ← 这句不在循环里！只执行一次！
```

**应对**：**养成好习惯：写 for 时，先把大括号 `{ }` 打好，再往里面填代码。**

## 九、下节课预告

> 下节课我们学 **while 循环** —— 另一种让计算机"重复跑圈"的方式。
>
> for 循环像是"跑固定圈数"，你提前知道要跑多少圈；
> 而 while 循环像是"跑到没力气为止"，你不知道要跑多少圈，但你知道**什么时候该停**。
>
> 我们还会学两个"特殊指令"：
> - **break**：跑到一半突然说"我不跑了！"直接停下。
> - **continue**：这一圈不想做了，**跳过**这一圈，直接开始下一圈。
>
> 很有意思的，期待下节课见！😊
