---

### 4.2 讲解① 变量——贴了标签的盒子（15 min）

#### （1）引入比喻

> 想象你有一个**盒子** 🎁。
>
> - 盒子上贴了一张**标签**，写着"年龄"。
> - 盒子里装了一个数字：**10**。
> - 以后你想知道年龄是多少，只要打开"年龄"这个盒子看一看就行了。
> - 而且你还可以把盒子里的东西**换掉**——今年是 10，明年就换成 11。
>
> 在 C++ 里，这个盒子就叫做**变量**。标签就是**变量名**，盒子里的数字就是**变量的值**。

#### （2）创建一个盒子——声明变量

```cpp
int age;       // 创建了一个盒子，标签叫 age，盒子里还没放东西
```

> `int` 是英语 integer（整数）的缩写，它告诉电脑：**这个盒子只能装整数**。就像有些收纳盒上写着"只放袜子"一样，`int` 的盒子只能放整数。

#### （3）往盒子里放东西——赋值

```cpp
int age;
age = 10;      // 把 10 放进 age 这个盒子里
```

> `=` 在 C++ 里不是"等于"，而是**"把右边的东西塞进左边的盒子"**。可以读作"赋值为"。

也可以创建盒子的同时就放东西进去：

```cpp
int age = 10;  // 创建盒子的同时放入 10
```

#### （4）看看盒子里有什么——输出变量

```cpp
cout << age;   // 注意！这里不加引号
```

> **关键区分**：
> - `cout << "age";` → 屏幕显示的是三个字母 a-g-e（因为加了引号，是文字原样输出）
> - `cout << age;` → 屏幕显示的是 10（不加引号，电脑会打开盒子看看里面装了什么）
>
> 比喻：加引号就像你对电脑说"请念出 age 这三个字母"；不加引号就像你说"请打开名叫 age 的盒子，告诉我里面是啥"。

#### （5）盒子里的东西可以换

```cpp
int age = 10;
cout << age << endl;   // 输出 10
age = 11;              // 换成 11
cout << age << endl;   // 输出 11
```

> 变量，变量，**会变的量**。盒子里的东西随时都可以换新的。

---

### 4.3 讲解② `cin >>` 输入——电脑的耳朵（10 min）

> 上节课我们学了 `cout`，它是电脑的**嘴巴**，负责往屏幕上"说话"。
>
> 今天的新朋友叫 `cin`（念"C-in"），它是电脑的**耳朵** 👂，能从键盘**听到**你输入的内容，然后把它放进盒子里。

#### 基本语法

```cpp
int age;
cin >> age;    // 等你从键盘输入一个数，放进 age 盒子
```

#### 箭头方向口诀

> - `cout <<` 箭头 **朝左 ←**，信息从程序流向屏幕（"往外说"）
> - `cin >>`  箭头 **朝右 →**，信息从键盘流入程序（"往里听"）
>
> 记忆技巧：**`cin` 的箭头指向变量，表示"把数据送进盒子"；`cout` 的箭头远离变量，表示"把数据送出去"。**

#### 现场演示

在 Dev-C++ 中写出以下程序并运行，让同学们看到黑窗口等待输入的效果：

```cpp
#include <iostream>
using namespace std;
int main() {
    int age;
    cout << "请输入你的年龄：";
    cin >> age;
    cout << "你今年 " << age << " 岁！" << endl;
    return 0;
}
```

> 大家注意看——运行后，黑窗口停住了！它在等你输入。输入一个数字按回车，电脑就把它放进了 `age` 盒子，然后说出来。**这就是交互！你的程序第一次学会了"听话"。**

---

### 4.4 讲解③ 变量命名规则 + 字符串输入预告（15 min）

#### 变量名的规矩（简单版）

> 给盒子贴标签也不是随便贴的，有几条小规矩：
>
> 1. **只能用英文字母、数字和下划线 `_`**（不能用中文、不能用空格）
> 2. **不能以数字开头**（`2age` ❌，`age2` ✅）
> 3. **不能跟 C++ 的关键词撞名**（比如不能叫 `int`、`return`）
> 4. **大小写不同**（`Age` 和 `age` 是两个不同的盒子）
>
> 小技巧：取名字要有**意义**——装年龄就叫 `age`，装分数就叫 `score`，别都叫 `a`、`b`、`c`，不然以后你自己都忘了盒子里装的是什么！

#### 字符串类型小尝鲜（为成就感设计）

> 我们刚才的盒子只能装**整数**，如果想装一个**名字**（比如"小明"）呢？
>
> 有一种特殊的盒子叫 `string`（字符串），它可以装一串文字。我们今天先简单用一下，后面的课会详细讲。

```cpp
#include <iostream>
#include <string>
using namespace std;
int main() {
    string name;
    cout << "你叫什么名字？";
    cin >> name;
    cout << "你好，" << name << "！欢迎来到C++世界！" << endl;
    return 0;
}
```

> 运行一下——输入你的名字，电脑会跟你打招呼！是不是很酷？💪

## 五、课堂例题

### 例题 1：输入一个整数并输出（入门）

**题目**：请写一个程序，让用户输入一个整数，然后原样输出这个整数。

**讲解思路**：这是最简单的"一进一出"，帮学生建立 `cin` → 变量 → `cout` 的基本流程。

```cpp
#include <iostream>
using namespace std;
int main() {
    int a;
    cin >> a;
    cout << a << endl;
    return 0;
}
```

**带学生走一遍**：
1. 先声明一个盒子 `a`（`int a;`）
2. 用 `cin` 从键盘听一个数，放进盒子 `a`
3. 用 `cout` 把盒子 `a` 里的数说出来

**运行示例**：
```
输入：42
输出：42
```

---

### 例题 2：输入两个整数，输出它们的和（进阶一步）

**题目**：输入两个整数 a 和 b，输出它们的和。

**讲解思路**：
- 引入"两个盒子"的概念
- 演示 `cin >> a >> b;` 连续读入的写法
- 演示在 `cout` 中直接写表达式 `a + b`

```cpp
#include <iostream>
using namespace std;
int main() {
    int a, b;
    cin >> a >> b;
    cout << a + b << endl;
    return 0;
}
```

**带学生走一遍**：
1. `int a, b;` —— 一口气创建两个盒子（逗号隔开）
2. `cin >> a >> b;` —— 电脑先听一个数放进 `a`，再听一个数放进 `b`
3. `cout << a + b;` —— 电脑把两个盒子里的数加起来，把结果说出来

**运行示例**：
```
输入：3 5
输出：8
```

**补充讲解**：输入时两个数之间用空格或回车隔开都可以。

---

### 例题 3：输入名字，电脑打招呼（成就感设计）

**题目**：写一个程序，询问用户的名字，然后输出一句问候语。

**讲解思路**：
- 引入 `string` 类型（告诉学生这是"装文字的盒子"，今天先会用就行）
- 需要多加一行 `#include <string>`
- 体验"交互式程序"的乐趣

```cpp
#include <iostream>
#include <string>
using namespace std;
int main() {
    string name;
    cout << "请输入你的名字：";
    cin >> name;
    cout << "你好，" << name << "！很高兴认识你！" << endl;
    return 0;
}
```

**运行示例**：
```
请输入你的名字：Tom
你好，Tom！很高兴认识你！
```

**课堂互动**：让每个同学都输入自己的英文名字，看到电脑跟自己打招呼，激发成就感。

---

### 例题 4：输入三科成绩，计算总分（综合）

**题目**：输入语文、数学、英语三科成绩（整数），输出总分。

**讲解思路**：
- 巩固多个变量的声明
- 练习连续 `cin` 输入
- 引入"取有意义的变量名"的好习惯

```cpp
#include <iostream>
using namespace std;
int main() {
    int yu, shu, ying;
    cout << "请输入语文、数学、英语成绩：" << endl;
    cin >> yu >> shu >> ying;
    int total = yu + shu + ying;
    cout << "总分是：" << total << endl;
    return 0;
}
```

**带学生走一遍**：
1. 创建三个盒子：`yu`、`shu`、`ying`
2. 用 `cin` 一口气听三个数，分别放进三个盒子
3. 再创建一个盒子 `total`，把三个盒子的数加起来放进去
4. 用 `cout` 把 `total` 说出来

**运行示例**：
```
请输入语文、数学、英语成绩：
95 100 88
总分是：283
```

## 七、常见错误与应对

### 错误 1：`cin` 和 `cout` 的箭头方向写反

```cpp
// ❌ 错误
cin << a;      // 箭头方向反了
cout >> "你好"; // 箭头方向反了

// ✅ 正确
cin >> a;
cout << "你好";
```

**应对方法**：反复强调口诀——**"cin 往里收 `>>`，cout 往外送 `<<`"**。可以在黑板上画箭头示意：键盘 → → → 变量，变量 ← ← ← 屏幕。

---

### 错误 2：输出变量时多加了引号

```cpp
int age = 10;
cout << "age";  // ❌ 输出文字 age，不是数字 10
cout << age;    // ✅ 输出数字 10
```

**应对方法**：再次强调比喻——加引号是"念字母"，不加引号是"打开盒子看内容"。让学生两种都试一试，亲眼看到区别。

---

### 错误 3：忘记声明变量就使用

```cpp
// ❌ 错误
cin >> a;       // a 是什么？电脑不认识！

// ✅ 正确
int a;
cin >> a;
```

**应对方法**：比喻——"你得先去买一个盒子回来（声明），才能往里面放东西。没有盒子，东西放哪儿呢？"

---

### 错误 4：变量名以数字开头

```cpp
int 2a;    // ❌ 不能以数字开头
int a2;    // ✅ 可以
```

**应对方法**：告诉学生——"盒子的名字不能以数字开头，就像人的名字不会叫'3号张三'一样。"

---

### 错误 5：连续输入时忘记用多个 `>>`

```cpp
// ❌ 错误：只读了一个数
cin >> a;
// b 没有被读入！

// ✅ 正确：一次读两个
cin >> a >> b;
```

**应对方法**：强调 `>>` 就像传送带上的分拣口，每多一个 `>>` 就多听一个数。

---

### 错误 6：中英文标点混用

```cpp
int a；   // ❌ 中文分号
int a;    // ✅ 英文分号
```

**应对方法**：提醒学生写代码前切换到**英文输入法**。可以看状态栏——确保不是中文输入模式。如果编译报一堆莫名其妙的错误，第一件事就是检查有没有中文标点。
