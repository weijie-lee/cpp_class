| 项目 | 内容 |
|------|------|
| 课次 | 第 4 课 / 共 40 课 |

## 四、讲稿提纲

> 以下为教师口语化讲稿要点，请用四年级学生能听懂的语言讲授。

### 4.1 复习热身（5 min）

- "上节课我们认识了三种'盒子'——`int`（装整数）、`double`（装小数）、`char`（装一个字）。今天我们要学怎么让这些盒子里的数字'动起来'——做加减乘除！"

### 4.2 加减乘（10 min）

- **类比**：C++ 里的 `+`、`-` 和数学课完全一样。`*` 就是数学里的 `×`（因为键盘上没有 ×，所以程序员用星号 `*` 代替）。
- 现场演示：
  ```
  int a = 3, b = 5;
  cout << a + b;   // 8
  cout << a * b;   // 15
  ```
- 强调：**两个 `int` 相加/相减/相乘，结果还是 `int`。**

### 4.3 整数除法 vs 小数除法（15 min）⭐ 重点

- **生活比喻——分糖果**：
  - "7 颗糖平均分给 2 个人，每人拿几颗？" → 每人 3 颗，**剩下 1 颗没法再分**。
  - 在 C++ 里，`7 / 2` 就是问"每人拿几颗"，答案是 `3`，不是 `3.5`！
  - **口诀：整数除整数，小数点后面全扔掉（不四舍五入，直接砍）。**
- **想要 3.5 怎么办？**
  - 只要除号两边有一个是小数（`double`），结果就变成小数：
    ```
    cout << 7.0 / 2;    // 3.5
    cout << 7 / 2.0;    // 3.5
    ```
  - 比喻："只要有一个人说'我要精确分'，C++ 就会精确地算出小数。"
- **易混点对比表**（写板书）：

  | 表达式 | 结果 | 原因 |
  |--------|------|------|
  | `7 / 2` | `3` | 整数 ÷ 整数 → 整数（截断） |
  | `7.0 / 2` | `3.5` | 有小数参与 → 小数结果 |
  | `7 / 2.0` | `3.5` | 有小数参与 → 小数结果 |
  | `7.0 / 2.0` | `3.5` | 有小数参与 → 小数结果 |

### 4.4 取余运算 %（20 min）⭐ 重点

- **生活比喻——分糖果的"剩下的"**：
  - 刚才 7 颗糖分给 2 个人，每人 3 颗。那**剩几颗**？`7 % 2 = 1`。
  - `%` 读作"取余"或"取模"，就是做除法之后**剩下的零头**。
- **更多例子**（让学生口算抢答）：
  - `10 % 3 = ?`（10 ÷ 3 = 3 余 **1**）
  - `12 % 4 = ?`（12 ÷ 4 = 3 余 **0**，整除！）
  - `5 % 8 = ?`（5 ÷ 8 = 0 余 **5**，除不动的时候余数就是自己）
- **`%` 的三大妙用**（画在黑板上）：
  1. **取个位**：任何整数 `% 10` 就能拿到个位数字。例如 `123 % 10 = 3`。
  2. **取十位**：先 `/ 10` 再 `% 10`。例如 `123 / 10 = 12`，`12 % 10 = 2`。
  3. **判断整除**：如果 `a % b == 0`，说明 a 能被 b 整除。以后学了 `if` 就能用！
- **注意**：`%` 只能用在整数之间！`7.5 % 2` 会报错。

### 4.5 运算优先级（5 min）

- "和数学课一样：**先乘除，后加减**；想先算谁，就给谁加括号。"
- 例子：`2 + 3 * 4` → 先算 `3*4=12`，再算 `2+12=14`。
- `(2 + 3) * 4` → 先算 `2+3=5`，再算 `5*4=20`。
- "**拿不准的时候，多加括号准没错！**"

### 4.6 赋值运算符与复合表达式（5 min）

- **`=` 不是"等于"，而是"把右边装进左边的盒子"。**
  - `a = a + 1;` 意思是：把 a 的旧值拿出来加 1，再放回 a 这个盒子里。
- 复合表达式示例：
  ```cpp
  int price = 15;
  int count = 4;
  int total = price * count;   // 60
  int change = 100 - total;    // 40
  ```
- "表达式就像一道数学算式，C++ 会算出结果，然后你可以把结果存进一个变量。"

## 五、课堂例题

### 例题 1：计算器——基本四则运算（入门）

> 输入两个整数 a 和 b，输出它们的和、差、积、商和余数。

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;

    cout << a + b << endl;
    cout << a - b << endl;
    cout << a * b << endl;
    cout << a / b << endl;   // 整数除法
    cout << a % b << endl;   // 取余
    return 0;
}
```

**教师讲解要点**：输入 `7 2` 时，除法输出 `3`（不是 `3.5`），取余输出 `1`。让学生试不同数字感受规律。

---

### 例题 2：精确除法——小数结果（进阶）

> 输入两个整数 a 和 b，输出 a÷b 的精确小数结果。

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;

    // 方法：把 a 变成小数再除
    cout << 1.0 * a / b << endl;
    return 0;
}
```

**教师讲解要点**：`1.0 * a` 会把 `a` 从 `int` "升级"成 `double`，然后再除以 `b`，结果就带小数了。这是一个非常实用的技巧！

---

### 例题 3：拆分三位数（核心）

> 输入一个三位数，分别输出它的百位、十位和个位。
>
> 例如：输入 `527`，输出 `5 2 7`。

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int bai = n / 100;        // 527 / 100 = 5
    int shi = n / 10 % 10;    // 527 / 10 = 52, 52 % 10 = 2
    int ge  = n % 10;         // 527 % 10 = 7

    cout << bai << " " << shi << " " << ge << endl;
    return 0;
}
```

**教师讲解要点**：这道题综合运用了 `/` 和 `%`，是信息学竞赛的经典题型。带学生在草稿纸上手动演算一遍 `527` 的每一步。

---

### 例题 4：分钟转换为小时和分钟（生活应用）

> 输入一个分钟数 t（如 135），输出对应的小时数和剩余分钟数。
>
> 例如：输入 `135`，输出 `2 15`（即 2 小时 15 分钟）。

```cpp
#include <iostream>
using namespace std;

int main() {
    int t;
    cin >> t;

    int hours = t / 60;    // 135 / 60 = 2
    int mins  = t % 60;    // 135 % 60 = 15

    cout << hours << " " << mins << endl;
    return 0;
}
```

**教师讲解要点**：`/ 60` 取小时，`% 60` 取剩余分钟——这和分糖果的道理一模一样！让学生试试输入 `60`、`59`、`120` 等特殊值。

## 六、课堂练习题

### 练习 1：交换两个变量的值（基础）

> 输入两个整数 a 和 b，交换它们的值后输出。
>
> 提示：需要一个"临时盒子" `temp`。
>
> 输入示例：`3 5`
> 输出示例：`5 3`

### 练习 2：计算总价和找零（应用）

> 铅笔每支 3 元，橡皮每块 2 元。输入买铅笔的支数 a 和橡皮的块数 b，以及付的钱 money，输出总价和找零。
>
> 输入示例：`4 3 20`（4 支铅笔、3 块橡皮、付 20 元）
> 输出示例：`18 2`（总价 18 元、找零 2 元）

### 练习 3：秒数转换（进阶）

> 输入一个秒数 s（如 3661），输出对应的完整小时数、剩余分钟数和剩余秒数。
>
> 输入示例：`3661`
> 输出示例：`1 1 1`（1 小时 1 分钟 1 秒）
>
> 提示：先算小时（`/ 3600`），再算分钟（剩余秒数 `/ 60`），最后算秒（`% 60`）。

### 练习 4：判断奇偶——输出余数（预热）

> 输入一个整数 n，输出 n 除以 2 的余数（0 或 1）。
>
> 输入示例：`7`
> 输出示例：`1`
>
> （下节课学了 `if` 之后，我们就能输出"奇数"或"偶数"啦！）

## 七、常见错误与应对

| 序号 | 常见错误 | 错误示例 | 正确做法 | 教师应对策略 |
|------|---------|---------|---------|-------------|
| 1 | 以为 `7/2` 等于 `3.5` | `int a = 7/2;`（期望得到 3.5） | 整数除整数得整数；想要小数用 `7.0/2` | 反复用"分糖果"的比喻强化；让学生上机验证 |
| 2 | 把 `%` 用在小数上 | `double a = 7.5 % 2;` | `%` 只能用于两个整数之间 | 告诉学生"`%` 是分糖果的剩余，糖果必须是整颗的" |
| 3 | `=` 左边写了表达式 | `a + 1 = b;` | `=` 左边只能是一个变量：`b = a + 1;` | 比喻："盒子的名字写在左边，要装的东西写在右边" |
| 4 | 忘记运算优先级 | `2 + 3 * 4` 以为等于 `20` | 先乘除后加减，结果是 `14` | 提醒"和数学课一样"，拿不准就加括号 |
| 5 | 除以 0 | `cout << 10 / 0;` | 除数不能为 0，程序会崩溃 | 演示一次报错，告诉学生"0 不能当除数，数学课也不行" |
| 6 | 混淆 `/` 和 `%` | 想取余数却用了 `/` | `/` 是商，`%` 是余数——一个除法问两个问题 | 用"分糖果"每次同时问："每人几颗？剩几颗？" |
