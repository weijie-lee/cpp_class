### 阶段一：语法基础（第 1 — 12 课）

> 🎯 **核心能力**：能用 C++ 写出完整的、能运行的程序

| 课次 | 知识点 | 一句话总结 |
|------|--------|------------|
| 第1课 | Hello C++ | `#include <iostream>` + `using namespace std;` + `int main()` 是每个程序的"三件套" |
| 第2课 | 变量与输入 | 变量是"盒子"，`cin >>` 往盒子里装东西，`cout <<` 把东西拿出来看 |
| 第3课 | 数据类型 | `int`（整数）、`double`（小数）、`char`（字符）、`long long`（大整数） |
| 第4课 | 运算符与表达式 | `+` `-` `*` `/` `%`，注意整数除法会截断，`%` 取余很有用 |
| 第5课 | if 单双分支 | `if` 是"如果"，`else` 是"否则"——让程序学会做选择 |
| 第6课 | if 多分支 + 逻辑运算符 | `else if` 多条路，`&&`（并且）`||`（或者）`!`（取反） |
| 第7课 | for 循环 | `for(int i=0; i<n; i++)` ——让程序学会重复做事 |
| 第8课 | while 循环 + break/continue | `while` 更灵活，`break` 跳出，`continue` 跳过 |
| 第9课 | 循环嵌套 | 循环里面套循环——打印图案、遍历二维空间 |
| 第10课 | 一维数组定义与遍历 | 数组是"一排盒子"，下标从 0 开始 |
| 第11课 | 数组应用 | 求和、求最大值、计数、查找——数组的四大基本操作 |
| 第12课 | 阶段一复习与测试 | 语法基础大通关 |

**复习互动提问**：
- 🙋 C++ 程序的基本结构是什么？（三件套）
- 🙋 `17 / 5` 和 `17 % 5` 分别等于多少？（3 和 2）
- 🙋 数组下标从几开始？（0）

### 阶段二：基础算法（第 13 — 24 课）

> 🎯 **核心能力**：能用函数组织代码，掌握常见的基础算法思路

| 课次 | 知识点 | 一句话总结 |
|------|--------|------------|
| 第13课 | 函数入门 | 函数是"小助手"，把重复的代码打包起来，需要时"喊一声"就行 |
| 第14课 | 二维数组 | 二维数组是"表格"，`a[行][列]` 定位一个格子 |
| 第15课 | 枚举法（一） | "一个一个试"——暴力出奇迹！ |
| 第16课 | 枚举法（二） | 优化枚举范围，减少不必要的尝试 |
| 第17课 | 模拟法（一） | "题目怎么说，代码就怎么写"——忠实模拟过程 |
| 第18课 | 模拟法（二） | 复杂模拟：多步骤、多条件的流程控制 |
| 第19课 | 冒泡排序 | 相邻比较、交换，大的"冒泡"到后面 |
| 第20课 | 选择排序 + 插入排序 | 选择排序"选最小"，插入排序"插对位" |
| 第21课 | sort 排序 + 贪心初步 | `sort()` 一行搞定排序；贪心 = 每步选当前最优 |
| 第22课 | 贪心算法 | 经典贪心问题：排队接水、活动安排 |
| 第23课 | 高精度加减法 | 用数组模拟竖式计算，突破 `long long` 的限制 |
| 第24课 | 阶段二复习与测试 | 基础算法大通关 |

**复习互动提问**：
- 🙋 冒泡排序的核心思想是什么？（相邻比较交换）
- 🙋 贪心算法的核心思想是什么？（每一步选当前最优解）
- 🙋 为什么需要高精度？（数字太大，`long long` 装不下）

### 阶段三：数据结构（第 25 — 36 课）

> 🎯 **核心能力**：掌握常用数据结构，会用合适的"工具"解决问题

| 课次 | 知识点 | 一句话总结 |
|------|--------|------------|
| 第25课 | 字符串基础 | `string` 是字符数组的升级版，支持 `+`、`.length()`、`[]` |
| 第26课 | 字符串应用 | 回文判断、字符统计、子串查找 |
| 第27课 | 前缀和 | 提前算好"累加"，后面查区间和只需一次减法 —— O(1) 的魔法 |
| 第28课 | 差分 | 前缀和的"反操作"，区间修改从 O(n) 变 O(1) |
| 第29课 | 二分查找 | "猜数字"游戏——每次砍掉一半，效率极高 |
| 第30课 | 二分答案 | 把"求最优解"变成"判断可不可行"——二分的高级用法 |
| 第31课 | 递归入门 | 函数自己调用自己——"套娃"思想 |
| 第32课 | 递归应用 | 汉诺塔、全排列——递归的经典问题 |
| 第33课 | 栈 | 后进先出（LIFO）——像叠盘子 |
| 第34课 | 队列 | 先进先出（FIFO）——像排队买饭 |
| 第35课 | vector + 结构体 | `vector` 是"会自动变长的数组"；`struct` 把多个变量打包 |
| 第36课 | 阶段三复习与测试 | 数据结构大通关 |

**复习互动提问**：
- 🙋 前缀和的作用是什么？（快速求区间和）
- 🙋 二分查找的前提条件是什么？（数据必须有序）
- 🙋 栈和队列的区别？（栈：后进先出；队列：先进先出）

### 阶段四：算法思想（第 37 — 39 课）

> 🎯 **核心能力**：理解搜索与动态规划思想，能解决更复杂的问题

| 课次 | 知识点 | 一句话总结 |
|------|--------|------------|
| 第37课 | DFS 深度优先搜索 | "一条路走到黑，走不通再回头"——回溯法的灵魂 |
| 第38课 | BFS 广度优先搜索 | "一圈一圈往外扩"——求最短路的好帮手 |
| 第39课 | 动态规划入门 | "记住算过的答案，避免重复计算"——用空间换时间 |

**复习互动提问**：
- 🙋 DFS 和 BFS 分别像什么？（DFS 像走迷宫；BFS 像水波扩散）
- 🙋 动态规划的核心是什么？（状态转移方程 + 记忆化）

## 六、教学备注

### 模拟赛组织注意事项

1. **考前**：提前打印试卷，准备草稿纸和铅笔
2. **考中**：保持安静，允许使用草稿纸，不允许交流
3. **考后**：快速批改（可互评），重点讲解错误率高的题目
4. **颁奖**：准备奖状或小奖品（哪怕是贴纸/糖果也很好！）

### 颁奖环节建议

- 播放一段轻松的背景音乐
- 每位同学都上台领取"结课证书"
- 拍集体合照留念
- 如果条件允许，可以准备一个小蛋糕庆祝 🎂
