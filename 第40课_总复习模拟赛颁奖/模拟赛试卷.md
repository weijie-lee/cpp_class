### 编程题 第 1 题（16 分）—— 入门

#### 甜甜的糖果

**题目描述**：

小明有 `a` 颗草莓糖和 `b` 颗薄荷糖。他想把所有糖果**平均分**给 `n` 个好朋友。

请你计算：
- 每个朋友能分到几颗糖？
- 分完之后还剩几颗糖？

**输入格式**：

一行三个正整数 `a`、`b`、`n`，用空格隔开。

**输出格式**：

一行两个整数，分别表示每人分到的糖果数和剩余的糖果数，用空格隔开。

**输入样例**：

```
10 7 4
```

**输出样例**：

```
4 1
```

**样例解释**：共 10+7=17 颗糖，17÷4=4 余 1，所以每人 4 颗，剩 1 颗。

**数据范围**：1 ≤ a, b ≤ 1000，1 ≤ n ≤ 100

### 编程题 第 2 题（16 分）—— 简单

#### 成绩等级

**题目描述**：

老师要给 `n` 个同学的成绩评等级。规则如下：

| 分数范围 | 等级 |
|----------|------|
| 90 — 100 | A |
| 80 — 89 | B |
| 60 — 79 | C |
| 0 — 59 | D |

请统计各等级的人数。

**输入格式**：

第二行 `n` 个整数，表示每个学生的成绩（0 ≤ 成绩 ≤ 100）。

**输出格式**：

一行四个整数，依次为 A、B、C、D 等级的人数，用空格隔开。

**输入样例**：

```
6
95 82 76 55 100 60
```

**输出样例**：

```
2 1 2 1
```

**样例解释**：
- A 等级（≥90）：95、100 → 2 人
- B 等级（80~89）：82 → 1 人
- C 等级（60~79）：76、60 → 2 人
- D 等级（<60）：55 → 1 人

**数据范围**：1 ≤ n ≤ 100，0 ≤ 成绩 ≤ 100

### 编程题 第 3 题（16 分）—— 中等

#### 前三名

**题目描述**：

班上有 `n` 个同学参加了编程比赛，每个同学有一个**姓名**和**分数**。

请你找出分数最高的**前 3 名**同学，按照分数**从高到低**输出他们的姓名和分数。

**输入格式**：

第一行一个正整数 `n`（3 ≤ n ≤ 100）。

接下来 `n` 行，每行一个字符串（姓名，不含空格）和一个整数（分数），用空格隔开。

**输出格式**：


**输出样例**：

```
Cathy 95
Alice 90
David 90
```

**样例解释**：分数排序后，第1名 Cathy（95），第2名 Alice（90），第3名 David（90）。Alice 和 David 同分，Alice 先输入所以排在前面。

**数据范围**：3 ≤ n ≤ 100，0 ≤ 分数 ≤ 1000

### 编程题 第 4 题（16 分）—— 挑战

#### 迷宫最短路

**题目描述**：

有一个 `n × m` 的迷宫，用 `.` 表示可以走的路，用 `#` 表示墙壁。

小明从左上角 `(1, 1)` 出发，想要到达右下角 `(n, m)`。

他每次可以向**上、下、左、右**四个方向走一步。

请问他**最少**需要走几步才能到达终点？如果无法到达，输出 `-1`。

**输入格式**：

第一行两个正整数 `n`、`m`，表示迷宫的行数和列数。

接下来 `n` 行，每行一个长度为 `m` 的字符串，表示迷宫。

保证 `(1,1)` 和 `(n,m)` 都是 `.`。

**输入样例**：

```
4 5
.....
.###.
...#.
.#...
```

**输出样例**：

```
8
```

**样例解释**：

```
S → → ↓ .
.  # # # ↓
.  . . # ↓
.  # → → E

路径：(1,1)→(1,2)→(1,3)→(2,3 不通)
实际路径：(1,1)→(1,2)→(1,3)→(1,4)→(1,5)→(2,5)→(3,5)→(3,4 不通)
最短路径步数为 8
```

**数据范围**：2 ≤ n, m ≤ 50

**提示**：这是一道经典的 BFS（广度优先搜索）问题。用队列存储当前位置和步数，逐层扩展，第一次到达终点时的步数就是最短路。

### 改错题 第 1 题 答案

**错误 ①**：`int sum;` → `int sum = 0;`

说明：变量未初始化，`sum` 的初始值不确定，会导致结果错误。（4 分）

**错误 ②**：`if (i % 2 == 1)` → `if (i % 2 == 0)`

说明：题目要求求偶数的和，`i % 2 == 1` 判断的是奇数，应改为 `i % 2 == 0`。（4 分）

---

### 改错题 第 2 题 答案

**错误 ①**：`for (int i = 1; i <= 5; i++)` → `for (int i = 0; i < 5; i++)`

说明：数组下标从 0 开始，`a[5]` 的有效下标是 0~4，`i=5` 时会越界。（4 分）

**错误 ②**：`int mx = 0;` → `int mx = a[0];`

说明：如果所有数都是负数，`mx = 0` 会导致结果错误。应该用数组第一个元素初始化。（4 分）

> 注意：第 ② 处错误改为 `int mx = a[0];` 后，下方循环的 `for (int i = 1; i < 5; i++)` 就是正确的（从第 2 个元素开始比较）。如果学生把循环也改为 `i = 0` 开始同样给分。

```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

struct Student {
    string name;
    int score;
    int id;  // 输入顺序
};

bool cmp(Student a, Student b) {
    if (a.score != b.score) return a.score > b.score;
    return a.id < b.id;  // 分数相同，按输入顺序
}

int main() {
    int n;
    cin >> n;
    Student stu[105];
    for (int i = 0; i < n; i++) {
        cin >> stu[i].name >> stu[i].score;
        stu[i].id = i;
    }
    sort(stu, stu + n, cmp);
    for (int i = 0; i < 3; i++) {
        cout << stu[i].name << " " << stu[i].score << endl;
    }
    return 0;
}
```

### 编程题 第 4 题 参考答案（16 分）

```cpp
#include <iostream>
#include <queue>
#include <cstring>
using namespace std;

int n, m;
char maze[55][55];
int dist[55][55];
int dx[4] = {-1, 1, 0, 0};
int dy[4] = {0, 0, -1, 1};

int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++) {
            cin >> maze[i][j];
        }
    }
    
    memset(dist, -1, sizeof(dist));
    
    queue<pair<int, int>> q;
    q.push({1, 1});
    dist[1][1] = 0;
    
    while (!q.empty()) {
        int x = q.front().first;
        int y = q.front().second;
        q.pop();
        
        for (int d = 0; d < 4; d++) {
            int nx = x + dx[d];
            int ny = y + dy[d];
            if (nx >= 1 && nx <= n && ny >= 1 && ny <= m
                && maze[nx][ny] == '.' && dist[nx][ny] == -1) {
                dist[nx][ny] = dist[x][y] + 1;
                q.push({nx, ny});
            }
        }
    }
    
    cout << dist[n][m] << endl;
    return 0;
}
```
