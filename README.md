# C++ 信息学启蒙课程 · 小学四年级

面向**四年级零基础**学生的完整 C++ 入门课程，目标冲击 **CSP-J / 蓝桥杯少儿组**。

- **课程规模**：40 课 × 100 分钟
- **授课节奏**：每周 1 次，讲练结合
- **开发环境**：Dev-C++（Windows）
- **课后作业**：每课 1-2 道洛谷题，不搞题海

## 课程结构（四阶段）

| 阶段 | 课次 | 主题 |
|------|------|------|
| 一、语法启蒙 | 第 01 - 12 课 | 输入输出 → 变量 → 数据类型 → 运算符 → 分支 → 循环 → 数组 → **阶段测试** |
| 二、基础算法 | 第 13 - 24 课 | 函数 → 二维数组 → 枚举 → 模拟 → 排序 → 贪心 → 高精度 → **阶段测试** |
| 三、数据结构 | 第 25 - 34 课 | 字符串 → 前缀和差分 → 二分 → 递归 → 栈队列 → STL → 结构体 → **阶段测试** |
| 四、算法思想 | 第 35 - 40 课 | DFS → BFS → DP（选学） → **总复习 + 模拟赛** |

## 每课文件夹结构

```
第XX课_主题/
├── 教案.md           教学文档（9 段式：信息/重难点/时间/讲稿/例题/练习/常见错误/作业/预告）
├── examples.cpp      所有例题代码（英文注释 + Expected Output）
├── practices.cpp     练习题参考答案
└── pdf/              A4 打印版（由 tools/build_pdf.py 生成）
    ├── 01_教案.pdf           教师讲稿（精简版）
    ├── 02_例题讲义.pdf        课堂发给学生的例题材料
    └── 03_练习册.pdf          学生填写册，含姓名栏 + 作答留白
```

**特殊课程**：
- 第 12 / 24 / 34 课：`测试卷.md` + `test_answers.cpp` + `pdf/` 内分为 复习 / 学生卷 / 答案
- 第 40 课：`模拟赛试卷.md` + `exam_answers.cpp` + `pdf/` 内分为 总复习 / 模拟赛 / 答案

## 打印使用

推荐直接使用 `pdf/` 目录里的 A4 PDF：

- **课前**：打印 `01_教案.pdf` 当作教师备课参考
- **课上**：`02_例题讲义.pdf` 人手一份，跟随讲解
- **练习**：`03_练习册.pdf` 发给学生在本子上做，答完收回批改

如需重新生成 PDF：

```bash
make install-deps      # 首次使用
make all               # 全量重建所有课
make lesson N=15       # 仅重建第 15 课
```

详见 [tools/README.md](./tools/README.md)。

## 教学风格

- 语言**亲切口语化**：变量 = 贴标签的盒子、数组 = 一排储物柜、循环 = 跑圈、递归 = 俄罗斯套娃
- **讲 25 + 练 10 + 讲 25 + 练 10 + 总结 25 + 休息 5** 的节奏，讲解段不超过 25 分钟
- **成就感设计**：每课结束学生应能拿出一个看得见的小成果
- **不讲**：指针、面向对象、引用传递、复杂模板

## 技术栈

- 源文件：Markdown（教案） + C++（代码）
- 打印版：WeasyPrint（HTML/CSS → PDF），支持中文字体、A4 分页、页眉页脚
