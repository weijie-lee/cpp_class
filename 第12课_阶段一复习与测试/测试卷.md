### 第 10 题 参考答案（22 分）

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    
    int a[110];  // 数组开大一点，保证够用
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    
    int maxVal = a[0];
    int minVal = a[0];
    
    for (int i = 1; i < n; i++) {
        if (a[i] > maxVal) {
            maxVal = a[i];
        }
        if (a[i] < minVal) {
            minVal = a[i];
        }
    }
    
    cout << maxVal - minVal << endl;
    
    return 0;
}
```

| 得分点 | 分值 |
|--------|------|
| 程序结构完整 | 2 分 |
| 正确读入 n | 2 分 |
| 正确使用数组存储 n 个数 | 4 分 |
| 正确求最大值（初始化 + 遍历比较） | 5 分 |
| 正确求最小值（初始化 + 遍历比较） | 5 分 |
| 正确输出最大值与最小值的差 | 4 分 |

> 💡 不使用数组、边读边比较的写法也正确，同样给满分：
