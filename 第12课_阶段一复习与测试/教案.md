### 专题一：输入输出（约 6 分钟）

#### 知识要点速查表

| 知识点 | 说明 | 示例 |
|--------|------|------|
| `#include <iostream>` | 引入输入输出功能 | 每个程序必写 |
| `using namespace std;` | 使用标准命名空间 | 每个程序必写 |
| `cout <<` | 输出到屏幕 | `cout << "Hello";` |
| `cin >>` | 从键盘读入 | `cin >> x;` |
| `endl` 或 `"\n"` | 换行 | `cout << x << endl;` |
| 连续输出 | 用多个 `<<` 连接 | `cout << a << " " << b;` |
| 连续输入 | 用多个 `>>` 连接 | `cin >> a >> b;` |

#### 快速回顾例子

```cpp
#include <iostream>
using namespace std;

int main() {
    int age;
    string name;
    cout << "请输入你的名字和年龄：" << endl;
    cin >> name >> age;
    cout << name << "同学今年" << age << "岁了！" << endl;
    return 0;
}
```

> 🎯 **提问互动**：`cout` 和 `cin` 的箭头方向分别是什么？为什么这样设计？
>
> 答：`cout <<` 箭头指向屏幕方向（数据从程序流向屏幕），`cin >>` 箭头指向变量方向（数据从键盘流向变量）。

### 专题二：变量与数据类型（约 6 分钟）

#### 知识要点速查表

| 数据类型 | 用途 | 例子 | 占用空间 |
|----------|------|------|----------|
| `int` | 整数 | `int a = 10;` | 4 字节 |
| `double` | 小数 | `double pi = 3.14;` | 8 字节 |
| `char` | 单个字符 | `char c = 'A';` | 1 字节 |
| `string` | 字符串 | `string s = "hello";` | 不固定 |
| `bool` | 真/假 | `bool flag = true;` | 1 字节 |

#### 重要规则提醒

| 规则 | 正确 ✅ | 错误 ❌ |
|------|---------|---------|
| 变量先声明再使用 | `int x; cin >> x;` | `cin >> x; int x;` |
| 变量名不能数字开头 | `int score1;` | `int 1score;` |
| 整数除法会截断 | `7 / 2 = 3` | 不会得到 3.5 |
| char 用单引号 | `char c = 'A';` | `char c = "A";` |

#### 快速回顾例子

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 7, b = 2;
    cout << a / b << endl;      // 输出 3（整数除法，截断小数）
    cout << a * 1.0 / b << endl; // 输出 3.5（变成小数除法的技巧）
    return 0;
}
```

> 🎯 **提问互动**：`7 / 2` 和 `7.0 / 2` 的结果分别是什么？为什么不一样？

### 专题三：运算符与表达式（约 6 分钟）

#### 知识要点速查表

| 类别 | 运算符 | 说明 |
|------|--------|------|
| 算术运算 | `+  -  *  /  %` | 加 减 乘 除 取余 |
| 赋值运算 | `=  +=  -=  *=  /=` | `a += 3` 等价于 `a = a + 3` |
| 自增自减 | `++  --` | `a++` 等价于 `a = a + 1` |
| 关系运算 | `==  !=  <  >  <=  >=` | 结果为 true 或 false |
| 逻辑运算 | `&&  \|\|  !` | 与、或、非 |

#### 易错点重点强调 ⚠️

```
⚠️ 赋值 = 和相等 == 千万别搞混！
   a = 5   → 把 5 赋值给 a
   a == 5  → 判断 a 是否等于 5
```

#### 快速回顾例子

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 17;
    cout << a / 5 << endl;   // 输出 3 （商）
    cout << a % 5 << endl;   // 输出 2 （余数）
    
    a += 3;     // a 变成 20
    a++;        // a 变成 21
    cout << a << endl;  // 输出 21
    return 0;
}
```

> 🎯 **提问互动**：`%` 取余运算在什么场景下特别有用？
>
> 答：判断奇偶（`n % 2`）、判断能否整除、提取各位数字等。

### 专题四：分支结构（约 7 分钟）

#### 知识要点速查表

| 结构 | 格式 | 用途 |
|------|------|------|
| 单分支 | `if (条件) { ... }` | 满足就做，不满足就跳过 |
| 双分支 | `if (条件) { ... } else { ... }` | 二选一 |
| 多分支 | `if / else if / else` | 多选一 |
| 嵌套 | if 里面套 if | 更复杂的判断 |

#### 逻辑运算符搭配

| 运算符 | 含义 | 例子 |
|--------|------|------|
| `&&` | 并且（两个都要满足） | `age >= 6 && age <= 12` |
| `\|\|` | 或者（满足一个就行） | `day == 6 \|\| day == 7` |
| `!` | 取反 | `!(x > 0)` 等价于 `x <= 0` |

#### 快速回顾例子

```cpp
#include <iostream>
using namespace std;

int main() {
    int score;
    cin >> score;
    
    if (score >= 90) {
        cout << "优秀" << endl;
    } else if (score >= 80) {
        cout << "良好" << endl;
    } else if (score >= 60) {
        cout << "及格" << endl;
    } else {
        cout << "不及格" << endl;
    }
    return 0;
}
```

> 🎯 **提问互动**：如果输入 85，会输出什么？程序是怎么一步一步判断的？
>
> 答：先判断 `85 >= 90`？不满足 → 再判断 `85 >= 80`？满足！→ 输出"良好"，后面的不再判断。

### 专题五：循环（约 8 分钟）

#### 三种循环对比

| | `for` 循环 | `while` 循环 | `do-while` 循环 |
|---|-----------|-------------|----------------|
| **格式** | `for(初始;条件;更新)` | `while(条件) { ... }` | `do { ... } while(条件);` |
| **特点** | 次数明确时首选 | 条件控制，次数不定 | 至少执行一次 |
| **典型场景** | 从 1 加到 100 | 输入 0 停止 | 先做一次再判断 |

#### `break` 和 `continue`

| 关键字 | 作用 | 比喻 |
|--------|------|------|
| `break` | 立刻跳出整个循环 | "我不跑了，直接离开操场" |
| `continue` | 跳过本轮，进入下一轮 | "这圈不算，继续跑下一圈" |

#### 循环嵌套核心理解

```
外层循环控制"行"
内层循环控制"列"

外层每走 1 步 → 内层走完一整圈
```

#### 快速回顾例子 1：for 循环求和

```cpp
#include <iostream>
using namespace std;

int main() {
    int sum = 0;
    for (int i = 1; i <= 100; i++) {
        sum += i;
    }
    cout << "1+2+...+100 = " << sum << endl;  // 输出 5050
    return 0;
}
```

#### 快速回顾例子 2：嵌套循环打印九九乘法表（前 3 行）

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 3; i++) {       // 外层：行
        for (int j = 1; j <= i; j++) {   // 内层：列
            cout << j << "*" << i << "=" << i * j << "\t";
        }
        cout << endl;
    }
    return 0;
}
```

输出：
```
1*1=1
1*2=2   2*2=4
1*3=3   2*3=6   3*3=9
```

> 🎯 **提问互动**：外层 `i=2` 时，内层 `j` 从几跑到几？一共执行了几次？

### 专题六：一维数组（约 7 分钟）

#### 知识要点速查表

| 知识点 | 说明 | 示例 |
|--------|------|------|
| 声明 | `类型 数组名[大小];` | `int a[10];` |
| 下标 | 从 **0** 开始！ | `a[0]` 是第一个元素 |
| 赋值 | 逐个或循环赋值 | `a[0] = 5;` |
| 初始化 | 声明时给初值 | `int a[5] = {1,2,3,4,5};` |
| 遍历 | 配合 for 循环 | `for(int i=0; i<n; i++)` |

#### 常见数组操作

| 操作 | 思路 |
|------|------|
| 求和 | 遍历累加 `sum += a[i]` |
| 求最大值 | 先假设 `a[0]` 最大，逐个比较 |
| 求最小值 | 先假设 `a[0]` 最小，逐个比较 |
| 查找 | 遍历找目标值，找到输出下标 |
| 计数 | 遍历统计满足条件的个数 |
| 倒序输出 | `for(int i=n-1; i>=0; i--)` |

#### 快速回顾例子

```cpp
#include <iostream>
using namespace std;

int main() {
    int a[5] = {3, 8, 1, 6, 2};
    int maxVal = a[0];  // 先假设第一个最大
    
    for (int i = 1; i < 5; i++) {
        if (a[i] > maxVal) {
            maxVal = a[i];  // 发现更大的，更新
        }
    }
    cout << "最大值是：" << maxVal << endl;  // 输出 8
    return 0;
}
```

> 🎯 **提问互动**：如果数组有 100 个元素，找最大值需要比较多少次？
>
> 答：99 次（从下标 1 到 99，每个都和当前最大值比一次）。
