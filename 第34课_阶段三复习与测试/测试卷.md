## 一、读程写结果（每题6分，共30分）

### 第1题 答案（6分）

```

o
Wor

```

- `s.size()` → 10（1.5分）
- `s[4]` → `'o'`（`H-e-l-l-o`，下标0-4）（1.5分）
- `s.substr(5, 3)` → `"Wor"`（从下标5开始取3个字符）（1.5分）
- `t` = `"Hello" + " " + "World"` = `"Hello World"`（1.5分）

### 第2题 答案（6分）

```
2
8
```

**解析：** 这是斐波那契数列。

- `f(2) = f(1) + f(0) = 1 + 1 = 2`（3分）
- `f(3) = f(2) + f(1) = 2 + 1 = 3`
- `f(4) = f(3) + f(2) = 3 + 2 = 5`
- `f(5) = f(4) + f(3) = 5 + 3 = 8`（3分）

### 第3题 答案（6分）

```
30
3
```

**解析：**

- 依次 push：10, 20, 30, 40, 50（栈顶是50）
- pop 两次：弹出50，弹出40。栈顶变为30。（3分）
- `st.top()` → 30
- `st.size()` → 3（剩余 10, 20, 30）（3分）

### 第4题 答案（6分）

```

```

**解析：**

- 依次 push：3, 7, 1, 9（队首3，队尾9）

- `q.front()` → 1（2分）

- `q.size()` → 2（1分）

### 第5题 答案（6分）


## 二、改错题（每题10分，共20分）

### 第6题 答案（10分）

**错误1：** `while (left < right)` 应改为 `while (left <= right)`（5分）

> **原因：** 当 `left == right` 时，还有一个元素没检查，如果恰好是目标值就会漏掉。

**错误2：** `left = mid` 应改为 `left = mid + 1`（5分）

> **原因：** `a[mid]` 已经比 target 小了，下次搜索应该从 `mid + 1` 开始，否则可能死循环。

### 第7题 答案（10分）

**错误1：** `return a.score < b.score` 应改为 `return a.score > b.score`（5分）

> **原因：** 题目要求从高到低排序，应该是 `>` 不是 `<`。

**错误2：** `for (int i = 0; i <= 3; i++)` 应改为 `for (int i = 0; i < 3; i++)`（5分）

> **原因：** 数组只有3个元素（下标0、1、2），`i <= 3` 会访问 `stu[3]`，越界了。

### 第9题 参考答案（15分）

```cpp
#include <iostream>
#include <string>
#include <stack>
using namespace std;

int main() {
    string s;
    cin >> s;

    stack<char> st;
    bool ok = true;

    for (int i = 0; i < s.size(); i++) {
        if (s[i] == '(') {
            st.push('(');
        } else {  // s[i] == ')'
            if (st.empty()) {
                ok = false;
                break;
            }
            st.pop();
        }
    }

    if (ok && st.empty()) {
        cout << "Yes" << endl;
    } else {
        cout << "No" << endl;

    return 0;
}
```

### 第10题 参考答案（20分）

```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

struct Student {
    string name;
    int score;
    int id;  // 记录输入顺序，用于成绩相同时保持原序
};

bool cmp(Student a, Student b) {
    if (a.score != b.score) return a.score > b.score;
    return a.id < b.id;  // 成绩相同，按输入顺序
}

int main() {
    int n;
    cin >> n;

    Student stu[105];
    for (int i = 0; i < n; i++) {
        cin >> stu[i].name >> stu[i].score;
        stu[i].id = i;
    }

    sort(stu, stu + n, cmp);

    for (int i = 0; i < n; i++) {
        cout << stu[i].name << " " << stu[i].score << endl;
    }

    return 0;
}
```

**评分细则：**
- 正确定义结构体（3分）
- 正确读入数据（3分）
- 正确编写比较函数（按成绩从高到低）（4分）
- 正确处理成绩相同时保持原序（稳定排序）（4分）
- 正确调用 sort 排序（3分）
- 正确输出结果（3分）

> **注：** 如果学生没处理稳定排序（即成绩相同时没有保持原序），但其他部分正确，可给16分。使用 `stable_sort` 代替记录 id 的方式也可得满分。
